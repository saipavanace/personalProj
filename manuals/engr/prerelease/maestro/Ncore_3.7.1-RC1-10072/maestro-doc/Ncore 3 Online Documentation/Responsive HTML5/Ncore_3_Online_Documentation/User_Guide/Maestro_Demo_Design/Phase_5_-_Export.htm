8.6 Phase 5 - Export

Goals for this phase: Export RTL (refer to Export Design (Phase 5)).

1. Click Generate RTL.

2. Make selections for exporting the design (data set, output directory).

3. When finished, click Export.

4. In the Console, you should see a message indicating a successful export.

5. Save your design.
