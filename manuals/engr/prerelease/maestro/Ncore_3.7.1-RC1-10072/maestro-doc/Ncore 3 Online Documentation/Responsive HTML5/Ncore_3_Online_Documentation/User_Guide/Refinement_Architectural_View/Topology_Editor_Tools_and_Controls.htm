6.2 Topology Editor Tools and Controls

The Topology Editor provides tools and controls that you can use to edit the network diagrams.

6.2.1 Topology Editor

Use the Topology View Editor to select an object in the network diagram. After you select the object, the Parameter View displays the parameter names and values configured for the selected object. See Topology Editor Toolbar and Topology Editor Settings for details on using the toolbar and information on how to filter network elements of the design.

6.2.2 Parameter View

The Parameter View displays the parameter names and values defined for a selected object.

You can change the values for the selected object in the Parameter View.

6.2.3 Connectivity Table and Routing Table

The Connectivity Table for a topology editor can be viewed by toggling the Connectivity Table icon (). Each blue square in the connectivity table indicates an interconnection between two points.

To show the route of an interconnect, select a blue square in the connectivity table. A colored line shows both ends of the interconnect selected in the connectivity table. This will put the system into Select Routes mode. You can exit Select Routes mode using the Select Routes button in the toolbar.

You can also click a name on the side to highlight an entire row or column of connections at a time while inside the connectivity table.

The Routing Table can viewed by toggling the Routes Table icon (), which provides a table view of the routes in the network along with the elements of each routes. Selected routes are highlighted. Filtering can be applied to focus the table on the desired routes.
