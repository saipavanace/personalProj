7.1 Export a Design

To export a design, complete the following steps.

1. Review the console to ensure that no errors are displayed.

2. Fix any errors.

3. Navigate to Export Design and click Generate RTL.

4. The directory names, subdirectory names, file names, and tools contained in the Export Design panel are listed below:

- RTL

- Synthesis scripts   

- DCNXT

- RTLA

- Genus

- UVM testbench

- IPXACT

- SystemC models

5. If required, select Browse to change the default directory in which the Export Design package is placed. The default output path is the current working directory. The output file will be a zipped tar file with the current date and time as its name.

6. Select OK.
