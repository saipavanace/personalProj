2.1 The Maestro Workspace

The following figure shows a typical Maestro Workspace.

Key

Name

Description

A

Maestro Menus

Provides access to Maestro menus. See Maestro Menus and Shortcuts.

B

Horizontal Toolbar

Provides quick access to Maestro tools and functions. See Horizontal Toolbar.

C

Flow Navigator Ribbon

Shows the current phase of the Maestro workflow. See Flow Navigator Ribbon.

D

Project Tree pane

View the Project Tree of the database. The Project tab shows the name of the project. See Project Tree Pane.

E

Tasks View pane

View tasks within a Phase, which provide a workflow for completing the phase. See Tasks View Pane.

F

Object Editor pane

View the information about selected objects. See Object Editor pane.

G

Parameter View pane

View Maestro parameters. See Parameter View Pane.

H

Groups pane

Shows project groups. See Groups Pane.

I

Console pane

Command line and project log interface. Provides access to Issues, Help, and Global find. See Console Interface Pane and Other tabs.

J

Tiles pane

View and manipulate tile sets. A list of available tiles is shown in the pane. See Tiles Pane.

Dock Widgets

Panes can be undocked and moved using the widget in the upper right of each pane. Use the dock widget to restore the pane. Use X to delete the pane.
