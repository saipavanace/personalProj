Obtaining Maestro

Getting the Application

The Maestro application is typically distributed via the Arteris FTP server. An Arteris FAE or sales representative will provide the downloading instructions, user name, and password.

Getting a License File

You must obtain the license file from Arteris to use Maestro. Arteris uses the FlexLM license manager. Installation of the FlexLM software and the license file are described in Flex Licensing Manager.
