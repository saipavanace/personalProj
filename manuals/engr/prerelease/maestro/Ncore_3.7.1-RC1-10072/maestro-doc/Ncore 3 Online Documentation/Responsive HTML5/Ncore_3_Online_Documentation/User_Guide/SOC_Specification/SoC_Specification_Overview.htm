3.1 SoC Specification Overview

In Phase 1, you will create a chip and then specify the properties of the power and clock distribution within the chip. Maestro requires clearly defined relationships between how design components are connected to the power and clock networks.

There are two sets of tasks for this phase:

• SoC Specification/Domains (see “Domains”)

• SoC Specification/Safety (see “Setting the Safety Configuration”)
