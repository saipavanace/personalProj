2.5 Topology Editor Toolbar

The Topology Editor toolbar is visible when using the Topology Editor (Window > Views). For examples of using the Topology Editor, see Using the Topology Editor.

Key

Icon

Name

Description

A

Toggle Connectivity Table

Toggle connectivity table visibility.

B

Toggle Routes Table

Toggle routes table visibility.

C

Zoom In

Zoom window in.

D

Zoom Out

Zoom window out.

E

Zoom Selected

Fit view to selection.

F

Fit All

Fit view to all.

G

Relayout

Move from layout to grid view.

H

Distribute Adapters

Distribute adapters.

I

Prepend Adapter

Prepend an adapter to selected input.

J

Prepend Switch

Prepend switch to all inputs of the selected object.

K

Append Adapter

Append an adapter to selected output.

L

Append Switch

Append switch to all outputs of the selected object.

M

Insert Adapter

Insert an adapter to selected route segments.

N

Insert Switch

Insert a switch on selected route segments.

O

Split Switch

Split the selected switch.

P

Create Unconnected Switch

Create an unconnected switch.

Q

Delete Item

Delete selected item(s).

R

Merge Switch

Merge selected switches or toggle mode where switches can be merged by drag and drop.

S

Toggle Select Routes Mode

Toggle mode where routes are selected by clicking links.

Toggle Topology Configuration editor settings pane used to configure topology interface.

Toggle Topology Configuration editor settings pane used to configure topology interface.
