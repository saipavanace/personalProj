8.3 Phase 2 - System Assembly

Goals for this phase: Sockets, Memory Map, Communication (refer to System Assembly (Phase 2)).

1. Use the Tasks list to create two Initiator Sockets (click twice).

2. In the Project Tree select “default_initiator_socket”.

3. In the Parameter View update default_initiator_socket.

4. Change name to “caiu0”.

5. Toggle Enable poison bits on.

6. In the Project Tree select “default_initiator_socket_0”.
The default_initiator_socket_0 is displayed in the Parameter View.

7. In the Parameter View update default_initiator_socket_0.

a. Change name to “ncaiu0”.

b. Change the Protocol to AXI4.

c. Change AXI address bus width to 48.

8. Use the Tasks list to create a PCIe socket from the template.

9. Use the Tasks list to create 3 Memory Sockets (click 3 times).

10. In the Project Tree select default_memory_socket, default_memory_socket_0, and default_memory_socket_1.

11. Using the Object Editor:

a. Rename default_memory_socket to “to_delete”.

b. Rename default_memory_socket_0 to “dmi0” .

c. Rename default_memory_socket_1 to “to_delete_2”.
We will delete to_delete and to_delete_2 in a later step.

12. The Object Editor displays the updated values.

13. Use the Tasks list to create a Peripheral Socket.

14. Use the Project Tree to rename default_peripheral_socket to dii0.

15. The Project Tree displays the updated values.

16. Use the Tasks list to create an APB Socket for external access to the CSR network.

17. Use the Project Tree to rename default_external_config_socket to config0.

18. Use Tasks list to select “This Ncore Subsystem will have 2 NcoreDCEs”.

19. Use the Tasks list to navigate to System Assembly/Memory Map.

20. Use the Tasks list to create a Memory Map.

21. Use the Tasks list to Configure the DCE interleaving.

22. Click Save to configure DCE interleaving. Click on default_memory_map in the Project Tree to view the primary set select bits.

23. Use the Tasks list to create a Memory Set.
A memory set is created in default_memory_map.

24. Use the Tasks list to Partition MemorySet “default_memory_set into Dynamic Groups and Interleaving Functions”.

25. Use the Tasks list to navigate back to System Assembly/Sockets.
The project enters the System Assembly/Sockets stage.

26. Use the Object Editor to delete sockets to_delete and to_delete_2.
The sockets are removed from the project.

27. Undo twice to recover the missing sockets.
The sockets are reintroduced to the project 

28. Redo twice to remove the sockets.
The sockets are removed from the project.

29. Use the Tasks list to navigate to System Assembly/Memory Map.

30. The project enters System Assembly/Memory Map.

31. Use the Tasks list to create a Boot Region.

32. Use the Tasks list to create a Configuration Region.

33. Use the Tasks list to navigate to System Assembly/Communication.
The project enters System Assembly/Communication.

34. Use the Tasks list to navigate to Structural Design/Topology Structure.
