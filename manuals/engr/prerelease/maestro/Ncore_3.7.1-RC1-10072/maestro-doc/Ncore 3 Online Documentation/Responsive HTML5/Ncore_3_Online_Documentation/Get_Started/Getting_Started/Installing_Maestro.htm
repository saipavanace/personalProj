Installing Maestro

Maestro installation and administration is a simple process that installs Maestro software, Ncore package, documentation, and the Flexlm license manager along with their custom components and features as enabled by optional licenses.

Installation Path

The Maestro installer prompts you to install the product into a new directory in the current directory path. You can change the default installation path to one on a file server that may be accessible over a local area network.

Installation Procedure

To install Maestro:

1. Install package with shell, $ tar -xzvf <filename>.tar.gz

2. Set the following environment variables:
shell$ export MAESTRO_HOME <install_dir>
shell$ export NCORE3_MODEL_DIR=$MAESTRO_HOME/performance-modeling

3. export PATH=$MAESTRO_HOME/maestro-client/bin:$MAESTRO_HOME/maestro-server:${NCORE3_MODEL_DIR}/bin

4. Run a test to verify Maestro performance modeling is working:
ncore3_archsim --scenario-file $NCORE3_MODEL_DIR/test/regress/basic/basic_cnc1rd.scn
