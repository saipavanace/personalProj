5.5 Using the Topology Editor

This section provides examples of using Topology Editor.

5.5.1 Topology Editor Preferences

Clicking the gear in the upper right corner of the topology editor allows you to change visibility and whether to show labels for different objects. See Topology Editor Settings.

5.5.2 Switches

Creation: Right-click on canvas, and select Create Switch.


The newly created switch will appear on the canvas at the mouse location.


Double-click the switch to change its name.

5.5.3 Connectivity Table

Rows are Initiators. Columns are Targets.


Selecting a square will highlight the route in the Topology Window.

Selecting a row or column will select all routes on that row or column.

5.5.4 Route Editing

A purple selected route can be edited via drag-and-drop to move all the routes highlighted to the newly created switch.

Hint: all routes coming from/to an initiator/target must all go to the same first switch. After the first switch, they can diverge and go via different topologies.

5.5.5 Adapters

Width and clock adapters are automatically created using the “Insert required Adapters” from the task assistant. Pipe and rate adapters must be manually created.

Select the route segment to insert an adapter.

Select Insert Adapter from the toolbar, and select the adapter of choice. The name can be changed by double-clicking.

5.5.6 Clock Adapters

The clock domain for each topology object determines where clock adapters are required. To change the clock adapter locations simply, change the topology objects associated with the clock domain. This can be done by selecting the object of choice, then editing the domain in Parameter View.

After completion of manual topology editing, it is required to run the Insert Required Adapters task. This will ensure a legal topology.
