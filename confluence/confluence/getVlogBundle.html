Path

hw-lib/js/lib_utils.js

Description

This function returns an object (associative array) with each signal in the input signal bundle converted into a class VlogPort instance. The instances can be accessed with just signals names as keys. Expressions generated using these signals will automatically include name prefixes.

Usage

const myVlogBundle                  = getVlogBundle( { name: a string, signals: a signals bundle object } );

Return Value

An objects (associative array) of class VlogPort instances, with keys the same as keys of the signals parameter object.

Parameters

  1. name: Type: string. This is added as prefix to keys of parameter signals and used as port names when creating instances of class VlogPort.
  2. signals: Type: object. This is expected to be a signal bundle object ( such as returned from getSignalsBundle( ) method of a new style interface ). A signal bundle is simply a object of signal widths accessed with signal names as keys. Widths in a signal bundle can be negative or positive indicating direction.

Examples

// From new-style interfaces 
const instATP                              = new obj.userLib[ "InterfaceATP" ].                                            // Create an instance of interface of type ATP
const name                                 = "in_";
const signals                               = instATP.getSignalsBundle( { nVc: 1, wBus: 40, wProt: 0 } );       // Create a signals bundle object
const atpInterface                       = { name, signals };                                                                     // Old style interface
const inATP                                 = getVlogBundle( atpInterface );                                               // Convert old-style interface to a VlogBundle

inATP.bus                                             // Returns a VlogPort instance for bus.
\=inATP.bus=\                                     // Prints "in_bus" in TACHL.
\=inATP.bus.upper( 4 )=\                     // Prints "in_bus[ 39 : 36 ]
const fifoWdata                          = inATP.bus.upper( 4 ).cat( inATP.first, inATP.last );  // Returns an expression object (VlogExpr instance) of width 6 bits with first, last, and bus signals concatenated
\=fifoWdata=\                                     // Prints "{ bus[ 39 : 36 ], first, last }"
