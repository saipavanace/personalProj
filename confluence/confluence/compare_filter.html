Description

The block diagram is shown below:

The two inputs, in_a and in_b, are compared. When one changes from the other, a timer starts and the two inputs must return to be the same with in nCount clocks. If the timer ends while the two signals are different, error is asserted for one cycle.

The signal out is the last value that matched. This behavior is not affected by error or the timer. That is, if the two values match after the timer fires, signal out will change.
