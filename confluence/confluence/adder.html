SW Visible:

no

Parameters:

Javascript:

{
  "width" : integer, // Width of din and dout in bits, 2 or greater.
  "cs_width" : integer // Width of carry look ahead logic.
}

I/O:


u.port('input', 'a', width);   // a input
u.port('input', 'b', width);   // b input 
u.port('output', 's', width);  // s = a + b
u.port('output', 'co', width); // carry out of a + b

Modules Used:

None

Description:

This is an adder to use if having a problem with timing.

This block implements an adder where the carry tree (carry select style) width can be set. Each carry select group above the least significant group has two adds (one with carry in = 0 and one with carry in = 1. The add selected and sent to s is based on the carry in coming from below. The carry in sent to the next significant group is done in the same way.
