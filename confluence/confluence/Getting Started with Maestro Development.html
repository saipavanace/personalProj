Setup Local VM and Remote Server for Maestro Development

You can work on Maestro code, build it and test either locally (most developers use virtual machines) or remotely (on the Arteris server). It's advised to explore and be able to use both options.

For detailed instructions on setting up a VM (virtual machine) please refer to Creating a VM for Offline Maestro Development.

For information about remote development please refer to Setting Up Arteris Server for Maestro Development.


Build and Run Maestro

When your virtual machine and remote server are set up it's time to get Maestro code, build it and run. You can find all the information on Building and Running Maestro.


Get to Know Qt Creator

Maestro uses Qt toolkit to create graphical user interfaces (GUI). Even if don't work on GUI, you can still use Qt Creator IDE to browse code and build maestro. For details please refer to Using QtCreator to Edit and Debug Maestro.
