LATENCY CALCULATION: 

Read transaction: 

  Request path:

  AXI- AXI, APB-AXI, AXI-APB and APB-APB = Request cycle monitored at target - request cycle monitored at initiator. 


  Response path:

  AXI- AXI = Read data first cycle monitored at initiator - read data first cycle monitored at target.

  APB- AXI = Read data first cycle monitored at target - request cycle monitored at initiator

  APB-APB = Read data first cycle monitored at initiator - read data first cycle monitored at target.

  AXI-APB = Read data first cycle monitored at initiator - read data first cycle monitored at target.


  Roundtrip latecny:

  AXI-AXI, APB-AXI, AXI-APB, APB-APB: Read data first cycle monitored at initiator - Read request monitored at initiator.


Write transaction:

  Request path:

  AXI- AXI, APB-AXI, AXI-APB and APB-APB = Request cycle monitored at target - request cycle monitored at initiator. 

  Response path:

  AXI- AXI = write response cycle monitored at initiator - write response cycle monitored at target.

  APB- AXI =  write data first cycle monitored at initiator - write request cycle monitored at target.

  APB-APB = write data first cycle monitored at initiator - write request cycle monitored at target.

  AXI-APB =  write response cycle monitored at initiator - write data first cycle monitored at target.

  Roundtrip latecny:

  AXI-AXI, AXI-APB: Write response cycle monitored at initiator - write request monitored at initiator.

  APB-AXI, APB-APB: Write data first cycle monitored at initiator - write request monitored at initiator.
