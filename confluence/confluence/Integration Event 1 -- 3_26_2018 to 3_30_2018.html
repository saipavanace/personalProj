  • How to run legato top level simulation (with waveform)

    • Make a new work area 

% mkdir ie1_top

    • Go into your new work area 

% cd ie1_top

    • Run the setup script

% /engr/dev/tools/script/setup_sym -d master

    • Source the SOURCEME file 

% source SOURCEME.csh

    • Run a random simulation 

% node  $WORK_TOP/../../scripts/qrsim.js -e top_smi2 -c -n top_smi2_static -t pktizer_rand_test -s $WORK_TOP/tb/pktizer/switch_rtl_16x16.json,$WORK_TOP/tb/pktizer/top_wrapper_16x16.json,$WORK_TOP/../hw/blocks/smi_pkt/rtl/params/top_smi_16x16_cfg.json,$WORK_TOP/tb/pktizer/top_tb_16x16.json -q -p +en_dump,+NUM_TRANS=1000,+UVM_VERBOSITY=UVM_HIGH,+enable_slv_dly -d UVM_HIGH,CUSTOMER_TB -r "-qwavedb=+signal+memory+class+transaction+assertion" -R 1 -a
