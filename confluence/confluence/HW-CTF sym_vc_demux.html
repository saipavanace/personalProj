Module Name

sym_vc_demux

Parameters

{
  "interfaces: {
    "inInterface" : {InterfaceATP}, 
    "outInterfaces" : [{InterfaceATP},,,] // array of out interfaces
  "vcMap" : [{    // array of input VC to output port and and output port VC. length equals number of VCs on inInterface.
    "inVc" : an_integer,           // 0 to number of input VCs minus 1.
    "outInterface" : an_integer,   // 0 to number of outputInterfaces minus 1. The index number of outInterface in outInterfaces.
    "outVc" : an_integer},         // 0 to number of output VCs on that outInterface minus 1.
    {vc1Map}
    ...
    {vc\=inInterface.signals.valid-1=\Map}]
}

Description

Takes a single VC link and expands it to multiple links with a mapping of VCs. The most common use case is to map a VC link to multiple non-VC links.

An input VC can only map to one output interface.

Examples

Parameter values for a 3 VC link expanding to 3 non VC links:

{
 "interfaces" : {
   "inInterface" : {
     "name" : "in_",
     "direction" : "slave",
     "params" : {"file_path" : "$HWSYM_TOP/atp_params_vc3.json" },
     "interface" : "InterfaceATP",
     "pktDef": {
       "params" : {},
       "packet" : "PacketATP"
     }
   },
   "outInterfaces" : [
     {"name" : "out0_",
      "direction" : "master",
      "params" : {"file_path" : "$HWSYM_TOP/atp_params.json" },
      "interface" : "InterfaceATP",
      "pktDef": {
        "params" : {},
        "packet" : "PacketATP"
      }
     },
     {"name" : "out1_",
      "direction" : "master",
      "params" : {"file_path" : "$HWSYM_TOP/atp_params.json" },
      "interface" : "InterfaceATP",
      "pktDef": {
        "params" : {},
        "packet" : "PacketATP"
      }
     },
     {"name" : "out2_",
      "direction" : "master",
      "params" : {"file_path" : "$HWSYM_TOP/atp_params.json" },
      "interface" : "InterfaceATP",
      "pktDef": {
        "params" : {},
        "packet" : "PacketATP"
      }
     }
   ]
 },
 "vcMap" : [{"inVc" : 0, "outInterface" : 0, "outVc" : 0},
            {"inVc" : 1, "outInterface" : 1, "outVc" : 0},
            {"inVc" : 2, "outInterface" : 2, "outVc" : 0}]
}
