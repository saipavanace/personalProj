2-Port Memory

 

Description:  2-Port memory has 2 -ports (Port A is for write; Port B is for read), 2-clocks, 2 -Memory enables, 1-write enable, 2- addresses, 1-input data & 1-output data.

On CLKA, when MEA is high & WEA is high DA gets written into Port A.

on CLKB, when MEB is high QB gets read from Port B.

It is faster and smaller in area compared to Dual port memory of the same size.                                   

Pin Interface:

module sacrls0g4l2p256x122m4b1w0c1p0d0l0rm4rw00zh1h0ms0mg1 ( QB, ADRA, DA, WEA, MEA, CLKA, TEST1A, TEST_RNMA, RMEA, RMA, LS, ADRB, MEB, CLKB, TEST1B, RMEB, RMB);

parameter bits = 122, addrbits = 8;

// Outputs

output [bits-1:0] QB;

// Inputs

input [addrbits-1:0] ADRA;

input [bits-1:0] DA;

input WEA;

input MEA;

input CLKA;

input TEST1A;

input TEST_RNMA;

input RMEA;

input  [3:0] RMA;

input LS;

input [addrbits-1:0] ADRB;

input MEB;

input CLKB;

input TEST1B;

input RMEB;

input  [3:0] RMB;

 

Dual Port Memory       

Description: Dual Port memory has 2 -ports (Port A & Port B can be write/read), 2-clocks, 2 -Memory enables, 2-write enables, 2- addresses, 2-input data & 2-output data.

On CLKA, when MEA is high & WEA is high DA gets written into Port A; when WEA is low QA gets read from Port A.

Similarly, on CLKB, when MEB is high & WEB is high DB gets written into Port B; when WEB is low QB gets read from Port B.

It is slower and larger in area compared to 2-port memory of the same size.

 

Pin Interface:

module sadsls0g4u2p256x122m4b1w0c1p0d0l0rm4rw01zh1h0ms0mg1 ( QA, QB, ADRA, DA, WEA, MEA, CLKA, TEST1A, TEST_RNMA, RMEA, RMA, WA, WPULSE, LS, BC0, BC1, BC2, ADRB, DB, WEB, MEB, CLKB, TEST1B, TEST_RNMB, RMEB, RMB);

parameter bits = 122, addrbits = 8;
