Hycon:

  1. Update site.sh available in (/home/dbotero/workspace/VELOCE/HYCON/Arteris_Linux_01/config/site.sh)
  2. Download cross-compilation toolchain (available in free-electrons, and locally installed here: CROSS_TOOLCHAIN=/home/dbotero/workspace/VELOCE/HYCON/TOOLCHAIN/aarch64--glibc--stable-2018.11-1)
  3. Clone hycon platform:
    1. source site.sh
    2. hycon-clone --site-defaults site.sh $HYCON_HOME/platforms/multimedia/linux Arteris_Linux_01
    3. cd questa
    4. ./build
    5. ./run
  4. To rebuild Software:
    1. cd $HYCON_HOME/platforms/multimedia/linux Arteris_Linux_01/sw
    2. ./build devicetree-rebuild (need to have perl-devel package installed => yum)
