HOW TO RUN SIMULATIONS (ncore3.0.1):

  1. New checkout: /engr/dev/tools/script/setup_concerto

  2. source SOURCEME ncore3.0.1 (or source SOURCEME ncore3.2.x)

  3. export DESIGNWARE_HOME="/engr/dev/tools/synopsys/vip_amba_svt_S-2021.09-3-T-20211203"

  4. run command: runsim -e fsys_snps -c <config_name>_snps -t <test_name>
    example: runsim -e fsys_snps -c hw_cfg_7 -t chiaiu_random_read_snps

HOW TO RUN SIMULATIONS (ncore3.4):

  1. New checkout: /engr/dev/tools/script/setup_concerto

  2. source SOURCEME ncore3.4

  3. For VCS simulation only (runsim -sim VCS), add this step : module unload snps/2017.12_engineering;   module load snps/vcs/vU-2023.03;   export LM_LICENSE_FILE="5285@lic-node0" (in CSH: setenv LM_LICENSE_FILE "5285@lic-node0")

  4. Set $DESIGNWARE_HOME env variable (Path to svt amba vip. e.g. export DESIGNWARE_HOME="/engr/eda/tools/synopsys/vip_amba_svt_U-2023.03") : module load snps/vip/vU-2023.03

  5. Run fsys test,

    1. With Questasim simulator : runsim -e fsys_snps -c <config_name> -t <test_name>

    2. With VCS simulator, need to provide runsim switch - “-sim VCS“ to run with VCS : runsim -e fsys_snps -c <config_name> -t <test_name> -sim VCS

Note : Please don’t alter step-3 & step-4


HOW TO RUN SIMULATIONS (ncore3.6):

  1. New checkout: /engr/dev/tools/script/setup_concerto Ncore3.6/develop

  2. source hw-ncr/dv/scripts/setup_ncore36; cd $WORK_TOP

  3. For VCS simulation only (runsim -sim VCS), add this step : module unload snps/2017.12_engineering;   module load snps/vcs/vU-2023.03;   export LM_LICENSE_FILE="5285@lic-node0" (in CSH: setenv LM_LICENSE_FILE "5285@lic-node0")

  4. Set $DESIGNWARE_HOME env variable (Path to svt amba vip. e.g. export DESIGNWARE_HOME="/engr/eda/tools/synopsys/vip_amba_svt_U-2023.03") : module load snps/vip/vU-2023.03

  5. Run fsys test,

    1. With Questasim simulator : runsim -e fsys_snps -c <config_name> -t <test_name>

    2. With VCS simulator, need to provide runsim switch - “-sim VCS“ to run with VCS : runsim -e fsys_snps -c <config_name> -t <test_name> -sim VCS

Note : Please don’t alter step-3 & step-4
