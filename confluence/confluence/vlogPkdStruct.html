This is one of the JS functions returned by vlogGen() which can be found in the following path and file:
hw-lib/js/lib_utils.js

Base type

vlogSignal

Composition

class PkdStruct

Description

This is a factory function that returns a vlogSignal object represented as a packed struct. This means slices of the signal object can be accessed as members of a struct. The way to define a struct is explained here: Support for packed data-structures in TACHL. Each member of the struct can be of a different width, and a member can in turn be another packed struct or packed array definition.

Usage

The factory function can be called using the following call signature:

vlogPkdStruct( structDef, name,  options )

where structDef is an array param that defines the struct as explained in Support for packed data-structures in TACHL. The signal is represented by the string parameter name. And options is an optional parameter with the following properties:

  1. end: Specifies the end offset of the signal. If not specified, the end offset defaults to 0.
  2. endian: Specifies the endianness order in which the struct members are packed. Legal values: "little", "big". Default is "big" endian, which means that the first member (array index 0) occupies the most-significant bits.

Interface

This object returned by the vlogPkdStruct( ) factory function provides the following .

Data

The returned object will have the same data-properties as a vlogSignal object.

Methods

The returned object will provide the following methods: 

get ( key )

Return type: vlogSignal object. Returns a slice of the signal object that corresponds to the struct member whose name is specified by key (string). The returned vlogSignal object will also be of extended type vlogPkdStruct if the accessed member's packType = "struct", vlogPkdArray if the accessed member's packType = "array", or simply vlogSignal otherwise.

keys( )

Return type: iterator object. Returns an iterator to iterate through all the struct member names.

values( ) 

Return type: iterator object. Returns an iterator to iterate through all the struct members.

entries( )

Return type: Iterator object. Each next( ) call on the iterator returns an array with two elements, the first of which is the key / struct member name, and the second of which is the struct member (vlogSignal object).

Default iterator

Every vlogPkdStruct object has a default iterator which is same as the iterator returned by the function values( ) above. This allows the object to be used in the context of a for-of loop to iterate through all the struct members.
