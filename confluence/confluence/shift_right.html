Parameters:

Javascript:

{
  "width" : integer, // Width of din and dout in bits, 2 or greater.
  "minShiftSize" : integer // Minimum Write Size in bits of din, 1 or greater. Must be power of 2.
}

I/O:


ptr_width = log2ceil(width/minShiftSize);
u.port('input', 'din', width);        // Data in
u.port('output', 'shift', ptr_width); // Points at the last valid minWriteSize bit group on output 
u.port('output', 'dout', width);      // Data out

Modules Used:

None

Description:

This block implements a right shift with 0 fill.
