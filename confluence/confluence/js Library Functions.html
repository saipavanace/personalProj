testSymbolIndex(w,i)

Returns an object that gives properties about the symbol index i for an ecc code with ecc bits of width w.

{only_one_1,bit_set_even,bits_set}

only_one_1 means only 1 bit is set when the index is converted to binary.

bit_set_even means an even number of bits are set when the index is converted to binary

bits_set is an array of booleans than indicates which bits are set when the index is converted to binary

Coddington

parseVlogConst( x )

Returns an object with three properties: { 'width', 'radix', and 'value' }. The input parameter x is expected to be a

Verilog constant literal (string). The 'value' property is of type BigInt and the remaining properties are plain numbers.
