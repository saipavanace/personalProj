Descriptions

set_select is a module that select certain bits in an input address, as specified by a primary bit array parameter, and exclusive-OR each selected bit with every address bit as specified by a secondary bit array parameter.

Reference: Chapter 7 Selection Algorithms in Concerto System Parameter Specification Release 2.0

set_select is used in the Common Cache Pipe (CCP) that is a generic cache controller IP module used by several projects.
