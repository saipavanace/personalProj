The idea of this milestone is to support:

  • AXI, ACE, ACE-LIte, and ACE-LiteDVM Native Layer Connectivity Support

    • Excluding proxy cache so we arent dependent on Cache Core Wrapper yet

  • Running Multiple, Multi-Cacheline Transaction

    • Means Request and Response Ordering is Needed

  • WriteNoSnoop and ReadNoSnoop. Only connected to non coherent regions

  • Get Spyglass, Synthesis, and Coverage Running

  • Make sure Request ID flow is feasible.

  • Make sure Microarchitecture contains all needed info for verification of milestone.

  • The following fields will be fully supported . Otherwise field must be 0 or indicate no function / function should be disabled:

    • ID, Size, Len, Addr, Region, Burst, Prot, User, QoS, Loopback

    • AxCache = Modifiable/Non Modifiable

  • 64, 128, 256, and 512 Bus Widths

  • No Multi Port Support. (Only useful in AXI w Proxy Cache), however core is still stampable

  • At the end we will create base of check-in tests that must not be broken.
