Changes needed to enable reset looping in the current Symphony Testbench environment.

1.In the corresponding top  tb.json replace the current clock generator with "reset_if" as below.

"name" : "m_clk_rst_gen",

"module" : "tb_clk_rst_gen",

"name" : "reset_if",

"module" : "sym_reset_intf",

"moduleName" : "sym_reset_intf.v",

"tachlName" : "sym_reset_intf.v",

"localInterfaces" : ["a_"]

"moduleInterfaces" : ["a_"]

2.Add +arg for number of reset loops desired in the testcase (in runsim_testlist.json).

"plusargs":   

{  "+phase_loop" : "2" }

Eg:

$WORK_TOP/tb/pktizer/params/top_tb_16x16_cfg1.json

$WORK_TOP/scripts/runsim_testlist.json

Test invocation; runsim -t smi_loop
