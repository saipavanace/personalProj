New proposal for R1 (source : John Coddington (Deactivated)):

  • SW does not support the direct use of PMON probe and PMON stats blocks. This means there would be no need for SW to provide information about which PMON stats block and PMON probe blocks are associated with an ATU.

  • HW will integrate a stats block into both ATUIs and ATUTs to enable performance measurements. All registers associated with this function will become part of the ATU register description.

  • A single parameter will added to the ATUs that enables this function. The functionality will be fixed and not modifiable by the customer.
