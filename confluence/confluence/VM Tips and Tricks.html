Extend CentOS Storage

  1. In VirtualBox manager open  FIle → Virtual Media Manager
  2. Select your vdi image and resize it using a scroll in the bottom of the screen. Press Apply.
  3. Download latest gparted live cd image https://gparted.org/download.php and attach it to cdrom on your vm
  4. boot into gparted and resize system partition (ex. /dev/sda2), apply and reboot

  5. boot into centos and run

    sudo pvresize /dev/sda2
sudo lvresize /dev/mapper/centos-root /dev/sda2
sudo xfs_growfs / -d


Shared Folder

There are several options to use Shared Folders in Virtual box. This article describes one of the simplest.

Can't access to shared folder (permission)

If you have trouble with shared folder permissions : 
Edit the file /etc/group (you will need root privileges). Look for the line vboxsf:x:999 and add at the end :yourusername. To take effect you should log out and then log in.
