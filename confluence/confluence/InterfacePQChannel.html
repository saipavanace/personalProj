javascript parameters:

{ 
  "name": "a_string", 
  "params": 
     {
       wActive : integer , // Width of the active signal.
       channelType : string // Type of Channel ('P' or 'Q')

          },

  "direction" : "master" or "slave", 
  "interface": "InterfacePQChannel"
}

Signals:

output REQn; // if(channelType == Q) Indicates a request to change power states to the slave.
input ACCEPTn; // if(channelType == Q) Indicates a slave has accepted the power state change.
output REQ; // if(channelType == P) Indicates a request to change power states to the slave.
output STATE; // if(channelType == P) Indicates the power state to move to (0 - Off, 1 - On)
input  ACCEPT; // if(channelType == P) Indicates a slave has accepted the power state change.
input  [wActive-1:0] ACTIVE; // Indicates the slave is active
input  DENY; // Indicates a slave has denied the power state change.

Description:

This interface defines a P or Q-Channel connection to a downstream slave. This interface is derivative of the P/Q-Channel interface defined in IHI0068C_low_power_interface_spec.pdf (ARM P/Q Channel Spec)
