To enable squish gui tests in build process one must install squish package and define environmental variable SQUISH_INSTALL_DIR and SQUISH_LICENSEKEY_DIR

example for gr6-3:

export SQUISH_INSTALL_DIR=/engr/dev/tools/qt/squish-6.5.0-qt59x/squish-for-qt-6.5.0
export SQUISH_LICENSEKEY_DIR=/home/engadmin

It can be added to ~/.bashrc script

echo 'export SQUISH_INSTALL_DIR=/engr/dev/tools/qt/squish-6.5.0-qt59x/squish-for-qt-6.5.0' >> ~/.bashrc
echo 'export SQUISH_LICENSEKEY_DIR=/home/engadmin' >> ~/.bashrc

After that normal flow of running make cmake and make build can be executed and tests will be run together with make test command.


At the moment tests rely on dryrun.tcl for generation of project file.

One must comment out exit command at the end of the script for gui not to close before test finishes.
