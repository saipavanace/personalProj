HW Definition:

The Reorder Buffer Adapter is a bi-directional adapter and it can be used two different ways:

  1. Inside the fabric: ATP Links are used. Dependencies generated inside the block.
  2. Inside the CTL Block: SMI Links are used. Dependencies generated from outside the block.

Note: We are currently implementing the latter.

Conditions for adding a packet into the SRAM.

The key thing to remember, in order to add a packet into the SRAM, there must be a packet with similar characteristic already in progress, meaning::

  1. Same packet ID from the same initiator.
  2. Same channel (Read or Write) being accessed.
  3. Targeting different ATUt. 

Note: 

  • Under no conditions, the first packet of a given initiator will ever be stored.
  • Multicast is not supported for this R1 release.  (multicast →  One packet to multiple destinations).
