Overview

On AXI4 nativeInterface, coherency of a transaction is determined by the NC bit setting of the GPRAR the addresses maps to.

If NC=1, it is considered a non-coherent transaction.

If NC=0, it is considered a coherent transaction.

Boot-region: When boot-region access are made most likely the cache is disabled. So there is no allocation.

CSR region: CSR accesses are device, non-cacheable, non-bufferable accesses, so no cache allocation.

PCTCR.lookup/alloc_en should be default set as 0.
