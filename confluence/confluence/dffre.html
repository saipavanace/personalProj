Description 

This is an enable flip flop and supports following configurability

  1. Reset pin.  
  2. Width.

ParameterTypeRange of valuesDefault

Description

RstEnInteger0, 11Setting this parameter to 1 adds a reset pin. It only supports active low async reset.
