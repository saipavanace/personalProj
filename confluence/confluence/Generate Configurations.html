Purpose

This tool is meant to generate configurations from run_maestro on the queue and copy them to the directories in perf-vc/tests/json where tests expect them to be

  • maestro-examples contain jsons generated directly from MAESTRO_EXAMPLES
  • hw-tests-projects contain jsons generated from latest hw-tests-projects link

A set of config names to generate can be passed on the command line. They should match the name in MAESTRO_EXAMPLES, for example hw_config_01, and the name will be adjusted if it is in hw-test-projects, for example hw_cfg_1.

The commands that gen_cfg uses to generate the configuration are the same as inÂ Generate Maestro Jsons, the gen_cfg just provides a way to generate multiple ones in parallel on the queue and copy them to the appropriate destination.
