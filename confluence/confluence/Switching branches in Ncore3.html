Steps to switch to NCore3_Coverage branch

  1. mkdir <name>
  2. cd <name>
  3. /engr/dev/tools/script/setup_concerto (After this script is executed, current branch will be master)
  4. source the SOURCEME file. This should set the WORK_TOP variable.
  5. cd $WORK_TOP/dv/scripts
  6. . setup_ncore3_coverage.bash ( For bash. Note the space after dot is important) or source setup_ncore3_coverage.csh( For csh )
  7. Now the current branch should be NCore3_Coverage branch.

Steps to switch to Ncore3 branch

  1. mkdir <name>
  2. cd <name>
  3. /engr/dev/tools/script/setup_concerto (After this script is executed, current branch will be master)
  4. source the SOURCEME file. This should set the WORK_TOP variable.
  5. cd $WORK_TOP/dv/scripts
  6. ./setup_ncore301
  7. Now the current branch should be NCore3 branch.


Some important commands-

  1. git branch - tells you which is the current branch
  2. git checkout <branch name> - can be used to switch branches.
