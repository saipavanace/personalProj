perf_vc repo is for verification components to run tests/regressions.

Contractors developing verification components will be checking their code in this repo.

Dependencies for this repo are osci-lib, tempo

Instructions on building this repo.

  • module load maestro
  • git clone git@gitlab.arteris.com:maestro-dev/packages/perf_vc.git
  • cd perf_vc
  • mkdir build; cd build
  • cmake ..
  • make -j
