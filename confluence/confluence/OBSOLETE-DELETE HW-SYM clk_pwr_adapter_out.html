module_name

clk_pwr_adapter_out

JavaScript parameters

async: A boolean. True or False. Set this to true if clocks in inClkInterface and outClkInterface are asynchronous with respect to each other.

aFifoDepth: A positive integer greater than 1. Must be a power of 2 if async is set to true. Sets the depth of the pipes in the async adapters inside.

jitterOn: A boolean. True or False. Only for simulation. True causes synchronizer cells to jitter one cycle.

respType: A string that can take only one of the following values: "presto", "ncore". Setting to "ncore" strips away some of the logic not required due to the absence of the respSInterface.

protectionStyle: protectionStyle

Interfaces

clkInterface: A slave interface of type InterfaceCLK.

reqInterface: A slave interface of type InterfaceATP. This is the ATP interface for the request path.

respInterface: A master interface of type InterfaceATP. This is the ATP interface for the response path.

clkPwrInterface: A slave interface of type InterfaceCPA. This is the CPA interface that connects to the clk_pwr_adapter_out block.

pmaControlInterface: A slave interface of type InterfacePMAControl. This is used to communicate to PMA slaves via the busy signal,  if there are pending transactions.

protInterface: (Optional interface). A master interface of type InterfacePROT.

Modules Used

gen_async_adapter and gen_vc_async_adapter
