This is one of the JS functions returned by vlogGen() which can be found in the following path and file:
hw-lib/js/lib_utils.js

Base type

None

Composition

vlogSignal

Description

This is a factory function that returns an array (unpacked) of vlogSignal objects that can be accessed using indexing. The way to define a packed array is explained here: Support for packed data-structures in TACHL. Every array member is assumed to be of the same width and type. The member can in turn be another packed struct, or packed array definition. If the length property in the arrayDef is an array of numbers instead of a single number, then they are interpreted as a multi-dimensional array. The implementation doesn't actually use Verilog arrays but individual bit-vectors to represent each array element.

Usage

The factory function can be called using the following call signature:

vlogArray( arrayDef, name,  options )

where arrayDef is an object param that defines the packed array as explained in Support for packed data-structures in TACHL. The signal is represented by the string parameter name. And options is an optional parameter with the following properties:

  1. end: Specifies the end offset of the signal. If not specified, the end offset defaults to 0.
  2. endian: Specifies the endianness order in which the struct members are packed. Legal values: "little", "big". Default is "little" endian, which means that the first member (array index 0) occupies the least-significant bits.

Interface

This object returned by the vlogArray( ) factory function provides the following .

Data

The returned object will have the same data-properties as a vlogSignal object. In addition it has the following properties:

  1. length: The number of elements in the array.

Methods

The returned object will provide the following methods: 

idx ( i )

Return type: vlogSignal object. Returns a slice of the signal object that corresponds to the slice at index i (number). The returned vlogSignal object will also be of extended type vlogPkdStruct if the accessed member's packType = "struct", vlogPkdArray if the accessed member's packType = "array", or simply vlogSignal otherwise.

values( ) 

Return type: iterator object. Returns an iterator to iterate through all the array elements.

entries( )

Return type: Iterator object. Each next( ) call on the iterator returns an array with two elements, the first of which is the index, and the second of which is the signal slice at that index (vlogSignal object).

declare ( type )

Return type: string. Returns the Verilog declaration of all the signals in the array. type parameter is optional and a string that can take only two legal values: "wire", "reg".

Default iterator

Every vlogArray object has a default iterator which is same as the iterator returned by the function values( ) above. This allows the object to be used in the context of a for-of loop to iterate through all the array elements.
