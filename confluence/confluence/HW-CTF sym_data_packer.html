module_name:

sym_data_packer

Features

  • Packing Data
    • A register that accumulates a beat worth of data before forwarding the beat.

Packer 

I/O

input                                               in_valid;

input [wData-1:0]                            in_data

input                                               in_last

output                                             in_ready;

input [wData-1:0]                            in_mask

output                                             out_valid;

output [wData-1:0]                          out_data

output                                              out_last

input                                                out_ready;

Parameters

{

     wData: 1 - inf, // Width of the data to be stored,

}

Description

The packer will take in valid data as indicated by in_mask. Once all bits are valid, the output will be valid.

The output can also be valid if it is indicated the last beat came in. The out_last output will also be asserted.

If in_mask is all 1's then this circuit will be bypassed and the output signals will be tied to input signals.
