This is a JS function found in the following path and file:
hw-lib/js/lib_utils.js

Description

This is a factory function that returns an object with all the library helper functions and factory functions that constitute the Verilog expression generator as properties of the returned object (module pattern).

Usage

The factory function can be called using the following call signature:

vlogGen( options )

where options is an optional parameter (type: object) that is reserved for future parameters.

Interface

This object returned by the vlogGen( ) factory function provides the following properties:

Data

The returned object will provide an object called vlogHelper. The vlogHelper object provides a bunch of helper functions to work with vlog* objects.

Methods

The returned object will provide the following factory functions: 

  1. vlogSignal
  2. vlogConst
  3. vlogPkdArray
  4. vlogArray
  5. vlogPkdStruct

It also returns some regular functions:

  1. getVlogBundle
  2. declareVlogBundle
