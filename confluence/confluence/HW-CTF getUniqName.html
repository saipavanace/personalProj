Usage:

var new_name = getUniqName(names, request_name);

returns a string that is unique to used as an interface name.

names: an array of strings holding previously used names;

request_name: string that contains the name wanted.

Description:

This function returns a unique name based on the request_name. If the request_name exists, it starts adding "n_", where n is 0 to infinity and returns the first one that doesn't collide against a name in names. The new name is then added to names.
