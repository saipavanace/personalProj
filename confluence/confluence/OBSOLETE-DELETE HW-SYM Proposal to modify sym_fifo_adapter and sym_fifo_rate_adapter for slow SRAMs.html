Parameters

Add parameters:

sramClkInterface:

A clk Interface into the SRAM block. Assumed to be in the same domain as the clkInterface and so synchronous.  Also assumed to be an integer multiple of clkInterface.

sramRatio:

The width ratio of the SRAM to the incoming data width. If data width coming in is 32 and the sramRatio os 2, then the data width of the SRAM is 64. This ratio should be equal to the (Frequency clkInterace)/(Frequency slowClkInterface). Parameter ignored if slowClkInterface '_SKIP_' is true.

Assumptions

When slowClkInteface and clkInterface are both active, clock interface signal edge_en is present.

A new block exists that is called sym_sync_adapter that synchronizes an ATP interface across two subdomain interfaces where one's frequency is a whole number multiple of the other.

The block that is presently sym_fifo_adapter becomes fifo_adapter.

The block that is presently sym_fifo_rate_adapter becomes fifo_rate_adapter.

The dw_adapters are configured with pipeForward = false, pipeBackward = true and default depths.
