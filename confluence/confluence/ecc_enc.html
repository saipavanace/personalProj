Description

Takes in data and generates ecc bits.

The code uses a Modified Hamming Code for the SECDED code. The data is organized as:

 

data[N-1:0]

ecc[M-1:0]

symbol[M+N-1:0]

symbol[0] = ecc[M-1]

symbol[1] = ecc[0] // 2^0 = 1

symbol[2] = ecc[1] // 2^1 = 2

symbol[3] = data[0]

symbol[4] = ecc[2] // 2^2 = 4

symbol[5] = data[1]

symbol[6] = data[2]

symbol[7] = data[3]

symbol[8] = ecc[3]. // 2^3 = 8

.

.

.

symbol[M+N-1] = data[N-1]

 

ecc[M-1] = ^symbol[M+N-1:1]

ecc[0] = ^{any symbol entry where binary bit 0 in index is set, except when only bit 0 is set}

ecc[1] = ^{any symbol entry where binary bit 1 in index is set, except when only bit 1 is set}

ecc[2] = ^{any symbol entry where binary bit 2 in index is set, except when only bit 2 is set}

.

.

.

ecc[M-2] = ^{any symbol entry where binary bit M-2 is set, except when only bit M-2 is set}
