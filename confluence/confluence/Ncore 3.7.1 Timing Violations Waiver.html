 Root Causes Analysis

Resiltech configuration:

  • All critical blocks are failing with small margin except IOAIU C/D. Most of these violations can be met by enabling ULVT.

  • Most of the failures are due to high LOLs.

  • DMI A/B memory violations can be fixable by enabling slow memory principle (multi-cycle not enabled for this run).

  • IOAIU A/B memory violations magnitude is small, they can be fixable with faster memories/ floorplan/ enabling ULVT/ different memory sizes.

  • DMI A & B are identical hence the timing numbers are same. Similarly, DMI C & D are identical.

hw_config_10_lite configuration:

  • All critical blocks are failing with small margin except DMI_A that has a WNS slightly above 100ps. Most of the violations can be met by enabling ULVT.

  • Most of the failures are due to high LOLs.

  • IO_AIU_G memory violations can be fixable by enabling slow memory principle (multi-cycle not enabled for this run).

  • DMI and DCE memory violations magnitude is small, they can be fixable with faster memories/ floorplan/ enabling ULVT/ different memory sizes.

 Outcome
