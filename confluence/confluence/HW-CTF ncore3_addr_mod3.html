module_name

ncore3_addr_mod3

Javascript Parameters

wOffset : the granularity of the address block is 2^wOffset bytes. Its value ranges from 6 to 12 which means 64 bytes to 4K bytes.

I/O Ports

obj.lib.port('input', ‘address’, 52);

obj.lib.port('output', ‘result’, 2);

Description

Ncore3 Address Modulo 3.

This module computes modulo 3 of address[51:wOffset].

Modulo 3 is implemented using a tree of small lookup tables and adders.
