How do we put TD (Trace & Debug) in the most quiescent state possible?

from Jason:

The default state is set to not trigger on anything.

Master initiated trace default is to ignore trigger bit on interface.

NCore initiated trace default is to never trigger.

When there are no triggers, there is no trace capture and no trace accumulation.
