javascript parameters:

{ 
  "name": "a_string", 
  "params": 
     {
       "wPsel": integer, // 1 or greater
       "wAddr": integer, // 1 or greater, up to 32
       "wProt": integer, // 0 or 3, 
       "wData": integer, // 1 or greater, up to 32
       "wStrb": integer, // 0 or wData/8, there is one strobe for each 8 bits of data: strb[n] corresponds to data[(8n+7):(8n)]
       "wPSlverr": integer // 0 or greater
     },
  "direction" : "master" or "slave", 
  "interface": "InterfaceAPB"

}

Signals:

InterfaceAPB:

output [wAddr-1:0]                paddr;
output [wPsel-1:0]                psel;
output                            penable;
output                            pwrite;
output [wProt-1:0]                pprot;
output [wData-1:0]                pwdata;
output [wStrb-1:0]                pstrb;
input                             pready;
input [wData-1:0]                 prdata;
input [wPSlverr-1:0]              pslverr;
Description:

The APB interface is a low-cost interface that is optimized for minimal power consumption and reduced interface complexity. This interface is derivative of the AMBA APB protocol version 2.0 defined in IHI0024C_amba_apb_protocol_spec.pdf.
