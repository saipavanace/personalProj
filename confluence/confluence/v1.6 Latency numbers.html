ACE AIU

  • AR->CmdReq: 1 cycle
  • AW->CmdReq: 3 cycles (W is back pressured by the AIU such that the last beat ends up getting accepted one cycle after the CmdReq is sent out. First W is sent ~7-8 cycles before the AW)
  • SFI SnpReq->AC: 1 cycle
  • CR->SFI SnpRsp: 2 cycles
  • CD-> SFI DtrReq: Immediately
  • STR+DTR->R: 4 cycles (DTRReq followed by STRReq)
  • STR+DTR->R: 3 cycles (STRReq followed by DTRReq) -> 1 cycle with latest RTL latency fixes on 9/1/2016
  • DTWRsp->B: 2 cycles

Additional notes from Mike:

All of these numbers look right to me.

  • Note that the STR+DTR->R latency goes down to 3 if the STR comes first, and to 2 after my fix is released (but will stay at 4 whenever STR comes last)
  • Note that the AW->CmdReq is 3 because this is an ‘ACE’ config which deprioritizes writes. This latency will be 2 if it is an ACE-LITE and this is not the first write. (I.e., arbiter is parked on Writes).

DCE

  • Cmd Req to First SnpReq               3-Cycles
  • CmdReq to MRDreq                       3-Cycles
  • MrdRsp to STRreq                           1-Cycle
  • All SnpRsp to STRreq                      1-Cycle


IO-$

 This numbers are for 128-bit Bus Width. With nSets=1 and nWay:13

  • AR -> R                                                          11 Cycles Read Hit latency
  • Read Req -> IOC Tag lookup                          3  Cycles
  • IOC Tag lookup hit -> IOC data lookup           3  Cycles
  • IOC data lookup on hit -> Read response       4  Cycles
  • Write Req -> IOC Tag lookup                          7  Cycles
  • IOC eviction -> DTWReq                                12 Cycles
  • (IOC eviction) DTWRsp -> UPDReq                 1 Cycle

 

DMI

SFI MRD -> AR req                                                                                      0 cycles

R data -> DTR req (with bypass i.e. no interleaving)                                    0 cycles

R data (last) -> DTR req (first beat) (without bypass i.e. with interleaving): 1 cycles

SFI DTW -> AW req                                                                                      0 cycles

SFI DTW -> W data                                                                                      0 cycles
