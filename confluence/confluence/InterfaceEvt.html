JavaScript Parameters

{ 
  "name": "a_string" that is used as the prefix for signals of the interface
  "evtName": "a_string" that is used for the event name.
  "params": 
     {
       "width": integer, // 1 or greater, up to 32
       "eAck": boolean, // true or false, default: false
     },
  "direction" : "master" or "slave", 
  "interface": "InterfaceEvt"
}

Signals:

output [width-1:0] val; // This signal specifies the number of times the event has occurred. For single-width event, a logic-high indicates that the event occurred at that clock edge.
input ack; // This signal exists only if the parameter eAck is true

Description:

This interface is used to carry an event. When used as a multi-bit event (width > 1), the val (value) signal specifies the number of times the event occurred. And the ack signal when received is used to clear the val signal to zero, indicating acknowledgement of the event by the block that received the event.
