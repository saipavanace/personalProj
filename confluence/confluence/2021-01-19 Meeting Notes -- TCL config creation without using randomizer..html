Agenda: How to create TCL config without using randomizer with striping enabled.

  1. Example file: Use dv/tcl_cfg/striping_custom/1_striping.tcl as example file.
  2. Adding initiators and targets:
    Add new initiators and targets under # PHASE 2 section of above file. 
    Eg: To add new initiator: "set atui_7 [create_axi_init_socket $subsystem atui_6 $clocksubdomain AXI4 40 8 8 64 4 8 4 0 false true true true true true true 16]"
  3. Add flow:
    Add flow between initiator and target under "#------ Create flows" section
    Eg: create_flow $subsystem "$subsystem/atui_7" "$subsystem/atut_6" "atui_4_to_atut_6"
  4. Add/Update memory map: under "#------ Create Memory Map" section add/update memory map.
  5. Adding Stripe group and targets per stripe group: 
    Get "# Striped memory group 0" section from base.tcl to create a stripe group and modify "#update_object -name $smg0 -type "physicalChannels" -value_list [list $subsystem/atuT_0 $subsystem/atuT_2]" to change number of targets in given stripe group
    Note: Targets grouped together should be in the same flow. i.e., If Targets are grouped together and Initiator wants to access the stripe group then flow between that initiator and all targets present in the group must be established.
  6. Add Topology: 
    Under "#--- router ----" section update/add routing information based on topology that needs to be created.
