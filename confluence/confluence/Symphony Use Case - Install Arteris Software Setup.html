Name: Install Arteris Software Setup

User Roles: Iggy_IT

State: New

Phase: unspecified

Goals:

Request, receive, and install the desired Symphony Software.

Prerequisites:

  • Contact information for Arteris Sales
  • Centos-7 Linux machine for license server
  • Centos-7 Linux machine for software installation (may be the same as the license server)
  • Adequate disk space (how much is that?)

Frequency: Once

Steps:

  1. Call Arteris Sales and arrange to evaluate / purchase a set of Arteris software
  2. AEs provide keys and software
  3. Install License Server and Licenses
  4. Install Software
  5. Run Installation Test

Results:

A fully functioning installation of the chosen Arteris software.
