AXI Master driver:

  • Can drive read and write transactions simultaneously on both Read and Write channels.
  • Supports inter/intra-packet delays.
  • Supports  AXI4/AXI5 features.

APB Slave driver:

  • Gives write and read responses.
  • Supports delayed response.

Scoreboard:

  • Compares request conversion(from initiator to target)
  • Compares request+response conversion(from target to initiator)
