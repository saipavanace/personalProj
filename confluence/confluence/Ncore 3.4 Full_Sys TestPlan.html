Owner transfer: When a proxy cache is snooped with a non-invalidating snoop then it must transfer ownership of the line to the snooper (other proxy cache). 

–Non-Invalidating snoop is any snoop which does not require the snooped proxy cache to go to invalid state. For example: read shared snoop 

Sharer promotion: If a cache line in the system is available only in shared state, i.e. none of the proxy caches have it in owner state; then DCE will promote one of the sharer proxy caches as the owner and read the data form it instead of from DMI.
