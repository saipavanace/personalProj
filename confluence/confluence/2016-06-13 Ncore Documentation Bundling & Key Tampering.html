Decision

  • For every release of FlexNoC, CAE team will provide a build of FN that excludes documentation.
    • The core FN installer does not include 3rd party / open source libraries; those are installed by a second package within the FN install utility.
    • The FN installer asks for a unique key to fingerprint the install for downstream traceability.  This key is validated against legal key values, and is then used to modify the install in a way that is recognizeable.
  • CAE team will provide a second installer that installs just the documentation.  Engineering team will not get this installer.
  • Engineering team will create a new install script that calls all three installers.
    • The documentation installer is optional
    • The FN installer runs first, and asks users for the unique key.
