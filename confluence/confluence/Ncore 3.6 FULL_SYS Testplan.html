Memory like structures (large register files & Flop arrays), for example the OTT/ATT or skid buffers shall use parity (3.6, 3.8)
o Pipelines, Queues and address Fifos shall use parity
o State-machines shall detect illegal (non-reachable) states and preferably encode state information
in a way to have all state encodings separated by a Hamming distance ≥ 2. One-hot encodings shall
verify that only one bit is set
o All CSR registers shall use parity protection

6.1 Event Messaging Abdelaziz El Hamadi

Ncore 3.6 will support events messaging system, which are an asynchronous messages that will be used by all agents to notify other agents.

For 3.6 DVE will be the events broadcaster instead of DCE ,also DII & DMI will be able to generate events as well.

6.12. DMI Scratchpad Interleaving Abdelaziz El Hamadi

When multiple DMIs are part of the same interleaving group, it is possible that some of the bits used for interleaving, which are typically address bits just above the cacheline offset, are also used in the scratchpad index. This has the consequence of reducing the usable size of the scratchpad.
