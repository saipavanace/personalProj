gen_aria_dc.js is a script that

(a) reads in Maestro-generated output/rtl/design/hierarchy.json file,

(b) generates aria scripts for top-level modules and top module

(c) creates and runs gen_dc.sh to generate dc_scripts_* directories

(d) creates run_dc.sh for user to run the synthesis for each of the top-level modules and the top module


(1)

$ARIA_HOME/resources/app/scripts/gen_aria_dc.js 

OR

/home/boon/gen_aria_dc.js


(2)

node /home/boon/gen_aria_dc.js -h

would display help as below:

gen_aria_dc.js Usage <-h -topoMode $tm -clockUncertainty $cu -ungroupStartLevel $ug -incrementalOpt $io -ulvtPercentage $up -maxPaths $mp -rtlWrapperDir $rwd -hardMacroDbs $hmd > 
     -topoMode:          Optional. Boolean.    Default false
     -clockUncertainty:  Optional. Fraction.   Default 0.0
     -ungroupStartLevel: Optional. Integer.    Default 2
     -incrementalOpt:    Optional. Boolean.    Default false
     -ulvtPercentage:    Optional. 0 to 100.   Default 0
     -maxPaths:          Optional. 1 to 20000. Default 10000
     -rtlWrapperDir:     Optional. Path.       Default "'./memories'"
     -hardMacroDbs:      Optional. Array of path filenames. Default "['./memories/libraries_ssgn/*_lib.db']"
     -h:                 Optional. Prints out this message


(3)

Command lines to generate hw_config_02 RTL and run synthesis scripts out-of-the-box, with external memories as empty black boxes.


cd $WORK_TOP; mkdir my_synthesis; cd my_synthesis

run_maestro -utd -ndj -sdj -g -hws $WORK_TOP/../hw-sym -hwl $WORK_TOP/../hw-lib -hwc $WORK_TOP/../hw-ccp -hwn $WORK_TOP/../concerto -cfg hw_config_02 -mut $WORK_TOP/../test_projects/ncore_v3.0_configs/fsys_config/hw_cfg_2

node /home/boon/gen_aria_dc.js

source ./run_dc.sh


(4)

Command lines to run generate hw_config_10 RTL and run synthesis scripts out-of-the-box, with external memories specified.

cd $WORK_TOP; mkdir my_synthesis; cd my_synthesis

run_maestro -ndj --sym_debug_json -utd --hw_ncr_path $WORK_TOP/../concerto --hw_sym_path $WORK_TOP/../hw-sym/ --hw_lib_path $WORK_TOP/../hw-lib/ --hw_ccp_path $WORK_TOP/../hw-ccp/ --gen_rtl -g -cfg hw_config_10 -mut $WORK_TOP/../test_projects/ncore_v3.0_configs/fsys_config/hw_config_10_synthesis/default_design

cp /home/boon/hw_config_10_synthesis16_QoS/memories .

node /home/boon/aria3/m-phys/scripts/gen_aria_dc.js -rtlWrapperDir "'./memories'" -hardMacroDbs "['./memories/libraries_ssgn/*_lib.db']"

source ./run_dc.sh
