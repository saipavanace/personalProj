1. Interrupt Monitor: Monitors interrupt signal every cycle and performs analysis port write when Interrupt signal goes high.
  2. Interrupt Sequencer: Sets Interrupt variable when analysis port write is observed. This sequencer is started at SYM TXN Virtual sequencer.
  3. Interrupt Sequence: Sequence starts Interrupt service routine task when Interrupt sequencer variable is set to 1.
  4. Interrupt service routine task:

          a. Read top interrupt accumulator block register.

          b. Deduce if block setting interrupt is lower level interrupt accumulator block or regular block eg: ATUI, PMON, Firewall etc.

          c. If it's regular block then clear the interrupt register of the given block. Else navigate further down till regular block is found.

          d. Clear corresponding register bit of all interrupt accumulator blocks in the given hierarchy .

          e. Since Interrupt service routine is inside forever loop go to step 3 and wait for sequencer interrupt variable to be set to 1.
