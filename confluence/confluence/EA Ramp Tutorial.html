Maestro Tcl Commands

 

The Linux commands to setup the Maestro environment:

  •  module purge
  •  module load dev/maestro_test/stable

When there is qualified maestro build (that is, after all regression tests pass), the environment will be using the new build. The user does not need to change their environment to get the latest qualified build in their next run.

 

To copy the "ea_ramp" design directory to your local directory:

  • cp -r $MAESTRO_EXAMPLES/ea_ramp .

 

To execute Maestro to generate RTL:

  • run_maestro -g -t ea_ramp.tcl 

 

The RTL files are located in the sub-directories of the "output" directory.
