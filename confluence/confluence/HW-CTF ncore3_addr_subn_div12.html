module_name

ncore3_addr_subn_div12

Javascript Parameters

wOffset : the granularity of the address block is 2^wOffset bytes. Its value ranges from 6 to 12 which means 64 bytes to 4K bytes.

nSubtract : the modulo 12 number to be subtracted from address[51:wOffset].

I/O Ports

obj.lib.port('input', ‘address’, 52);

obj.lib.port('output', ‘address_out’, 52);

Modules used

ncore3_addr_div3

 

Description

Ncore3 Address Subtract n then Divide by 12.

This module substracts n (a modulo 12 number) from address[51:wOffset], and then divide the result by 12.

The least significant wOffset bits of input address is passed through unmodified to output address.

The most significant bits above wOffset bits of output address is the division result.

Divide by 12 is achieved by using divide by 3, then shift right by 2 bits.
