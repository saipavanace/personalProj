SW visible:

no

parameters

java_script:

{
	"width" : integer 2 or greater (bit2cntRom must be less than 9)
}

I/O

input din[width-1:0];  

output cnt[log2ceil(width+1)-1:0];  

Description

The block takes in a vector of 1s and 0s and returns the count of the number of 1s in the vector.

bits2cntRom

bits2cntRom uses a brute force look up (ROM) to determine the count. It is only for widths between 2 and 8 bits and is very fast for timing.

bits2cntWt

bits2cntWt implements the function using a Wallace tree that's terminated by an adder. It's primary used for a large number of bits. However, if it is desired to call this for low bit counts, it can be called directly.

bits2cnt

When the widths are from 2 to 8, bits2cnt calls bits2cntRom. When the widths are 9 or above, bits2cnt calls bits2cntWt.
