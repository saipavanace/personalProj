Protocol Style Description

ready/valid: Valid from the master. Ready from the slave. When ready and valid assert in the same cycle a transfer has occurred. Signals doing this function may have other names.

handshake: Asynchronous by nature. A signal on the master changes, which in turn causes a signal on the slave to change and so on and so on.

pulse: The value is sampled per clock and has meaning.

level: When the value changes information is transferred. Can be synchronous or asynchronous.
