Summary:

  1. Timing is bad in all groups. Flop2flop is violating by ~300ps with LoL = 43
  2. LOL is also above threshold in most of the groups.
  3. Congestion is high
  4. No timing loops, ULVT < 5%, clock uncertainty = 15%


Analysis:

  1. When compared worst path of 5nm in 7nm, I noticed these …
    1. 5nm path has SVT cells  vs 7nm path has LVT and ULVT
    2. This path has LOL > 40 in both tech nodes
