This is an example change units / names /projects  as needed.

 

Open a new terminal and setup conductor environment 

source /engr/dev/tools/script/conductor.csh

 

Setup directory structure

mkdir Synth_Test

cd Synth_Test

/engr/dev/tools/script/setup_hw

<Copy your modified RTL in to hw folder and commit it.>

<run npm i in hw and then in codegen folder>

<clone ncore_synth in Synth_Test>

 

Generate RTL 

<in Synth_Test>

mkdir EvalProject 

cd codegen

bin/generatecli.js -c ../EvalProject/evalConfig_coh.v -t ../EvalProject/top.v -f ../EvalProject ../test_projects/synthesis_projects/v1.0_eval_config_1.2GHz.apf


setup synthesis environment

setenv SYNTHESIS_HOME /engr/dev/tools/synopsys/syn_vL-2016.03-SP1
setenv SNPSLMD_QUEUE true

setenv LM_LICENSE_FILE 5285@lic-node0.arteris.com:5285@lic-node1.arteris.com:5282@lic-node0.arteris.com:1717@lic-node0:1717@lic-node1


Setup tech files and memory models 

cd  EvalProject/synthesis

cp ../../ncore_synth/tech_files/tech_inf/v1.0_eval_config_1.2GHz_RVT.inf evalConfig_parameters.inf

change ARTERIS_TECH_DIR variable in modified evalConfig_parameters.inf to point to your working directory. 

Run synthesis 

cd EvalProject/synthesis/units/A72_20

qsub -V -cwd -j y -o synthesis.log -b y dc_shell-t -topo -f A72_20.tcl
