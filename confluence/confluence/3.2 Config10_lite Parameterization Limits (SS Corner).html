Config10_lite specifications and performance (SS Corner)

 

Overall Summary:

All blocks are within the margin to be built at the 1.6GHz** target with the exception of the ioaiu with 2MB proxy cache which is currently 1.36GHz**. Some timing optimizations and a 2k SRAM is needed to close timing of this instance.

 

Blocks builds:

  • Synthesis Tool = Design Compiler -topographical

  • TSMC 7nm

  • target freq = 1.6GHz 

  • Slow Corner reported: Slow process, 0.675v, 125C

  • 5% limit on use of ULVT cells

  • 15% clock uncertainty

 ** These were synthesized with DC_TOPO & available SRAM’s [Doesn’t factor slow SRAMs, PnR, CTS...]

IOAIU Summary: (no resiliency, QosEnabled)

  • Instance A-F are within a comfortable margin of the target frequency. A place&route tool such as Fusion Compiler should correlate closely given they use similar constraints and SRAMs.

  • The “g” instance has a 2MB proxy cache and is below target. A memory compiler that can build 2k srams would greatly help this block
