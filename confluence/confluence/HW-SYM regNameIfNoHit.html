Usage:

var a_boolean = regNameIfNoHit(names, request_name);

returns a boolean. The value is false if it collides with a previous name, and true if there is no collision. 

names: an array of strings holding previously used names;

request_name: string that contains the name wanted to be added to names.

Description:

This function returns a boolean based on whether the request_name collides against a value in names. If request_name doesn't collide, it returns true and adds request_name to names. If it collides, it returns false and doesn't adds request_name to names.
