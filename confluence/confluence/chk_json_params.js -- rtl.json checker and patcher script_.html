The chk_json_params.js is used to check the rtl.json and also create a patched file(resolving errors/missing parameters if any)


Script path:

$WORK_TOP/scripts/chk_json_params.js


To check for errors run:

node chk_json_params.js rtl.json 


To also generate the patched file in the same dir as rtl.json:

node chk_json_params.js rtl.json true


Eg, output:

node chk_json_params.js rtl.json true

will output:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

SUMMARY

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

MISSING PARAMS EXIST INSIDE THE JSON. SEE MESSAGES ABOVE.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

SUCCESSFULLY CREATED PATCHED JSON: rtl.json_PATCHED

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
