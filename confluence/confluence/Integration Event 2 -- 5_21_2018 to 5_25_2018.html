GOALS:

  1. All three groups (Ncore-HW, Symphony-HW & Maestro-SW) deliver some components which will work together and demo the integration process.
  2. We should be able to pass through approximately 10 transaction per block (AIUs, DCE, DMI & DII) per fabric (yellow, green and blue) successfully.
  3. Our top level scoreboard should be able verify the correctness of these transactions.

INTERDEPENDENCIES:

  1. Symphony-HW team will deliver all three generated (fixed topology) and verified fabrics with pre-defined SMI interfaces as described below. Legato_tops (with fixed topologies) will be generated by Symphony verification team (no Maestro-SW support is needed for IE2).
  2. Ncore-HW team will deliver will deliver all required blocks (AIUs, DCE, DMI & DII) with bring-up completed at block level.
  3. Maestro-SW team will deliver RTL_TOP generation tool to stitch Ncore_top together with Legato_tops (fixed topology).

TIMELINE:

  1. Our initial assessment was last week of April.
  2. From current status Legato should be able to finish on time.
  3. From Ncore-HW & Maestro-SW, Chirag, MK and Manadher can estimate the timeline.

How to run legato top level simulation (with waveform)

  • Make a new work area 

% mkdir ie2_legato_top

  • Go into your new work area 

% cd ie2_legato_top

  • Run the setup script

% /engr/dev/tools/script/setup_sym -d master

  • Source the SOURCEME file 

% source SOURCEME.csh

  • Run a random simulation (Yellow Fabric)

% node $WORK_TOP/../../scripts/qrsim.js -e top_smi2 -c -n top_smi2_static -t pktizer_rand_test -s $WORK_TOP/tb/pktizer/ie2/top_wrapper_7x7_hier_yellow.json,$WORK_TOP/tb/pktizer/ie2/top_tb_7x7_yellow.json,$WORK_TOP/../hw/blocks/smi_pkt/rtl/params/smi_ie2_no_dp_cfg.json -q -p +en_dump,+NUM_TRANS=1000,+UVM_VERBOSITY=UVM_HIGH,+enable_slv_dly -d UVM_HIGH,CUSTOMER_TB -r -qwavedb=+signal+memory+class+transaction+assertion -R 1 -a -T -j $WORK_TOP/scripts/dv_lib_utils.js

  • Run a random simulation (Green Fabric)

% node $WORK_TOP/../../scripts/qrsim.js -e top_smi2 -c -n top_smi2_static -t pktizer_rand_test -s $WORK_TOP/tb/pktizer/ie2/top_wrapper_5x5_green.json,$WORK_TOP/tb/pktizer/ie2/top_tb_5x5_green.json,$WORK_TOP/../hw/blocks/smi_pkt/rtl/params/smi_ie2_no_dp_green_cfg.json -q -p +en_dump,+NUM_TRANS=1000,+UVM_VERBOSITY=UVM_HIGH,+enable_slv_dly -d UVM_HIGH,CUSTOMER_TB -r -qwavedb=+signal+memory+class+transaction+assertion -R 1 -a -T -j $WORK_TOP/scripts/dv_lib_utils.js

  • Run a random simulation (Blue Fabric)

% node $WORK_TOP/../../scripts/qrsim.js -e top_smi2 -c -n top_smi2_static -t pktizer_rand_test -s $WORK_TOP/tb/pktizer/ie2/top_wrapper_4x4_blue.json,$WORK_TOP/tb/pktizer/ie2/top_tb_4x4_blue.json,$WORK_TOP/../hw/blocks/smi_pkt/rtl/params/smi_ie2_blue_cfg.json -q -p +en_dump,+NUM_TRANS=1000,+UVM_VERBOSITY=UVM_HIGH,+enable_slv_dly -d UVM_HIGH,CUSTOMER_TB -r "-qwavedb=+signal+memory+class+transaction+assertion" -R 1 -a -T -j $WORK_TOP/scripts/dv_lib_utils.js
