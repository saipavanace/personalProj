javascript parameters:

{ 
  "name": "a_string", 
  "params": { "width" : integer },
  "direction" : "slave" or "master", 
  "interface": "InterfaceINT"

}

Signals:

If "direction" is "slave": 
   input [width-1:0] interrupts;
   
If "direction" is "master": 
   output [width-1:0] interrupts;

 

Description:

Interrupt interface is an interrupt bit vector.
