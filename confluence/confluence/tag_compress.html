Descriptions

tag_compress is a module that filter out some specified address bits from the input address bits, so as to generate a tag.

The address bits that are filtered out are those in the cacheline offset address bits (least significant bits of input address), and those in the primary bit array.

tag_compress is used in the Common Cache Pipe (CCP) that is a generic cache controller IP module used by several projects.
