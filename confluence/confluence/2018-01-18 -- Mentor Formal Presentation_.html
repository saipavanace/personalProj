    Structural Analysis and Fault Campaigns

    Single-point faults in ISO26262 systems can be mitigated by safety systems or secondary safety systems which detect faults in the main system or safety system, respectively, and apply preventative output. Structural analysis can detect when outputs do not depend on one or more of these systems. If an output does not depend on enough of these, then it can harbor either a single fault or a latent fault and is in need of further analysis. More detailed properties can be written to investigate failure cases, but structural analysis can conclusively rule out broad classes of faults.

    The structural analysis and fault campaign management tools output a range of values for the percentage of single/double faults mitigated. Additional proof work can converge both the upper and lower bounds of this range.

    Autocheck

    Autocheck is Mentor's automatic formal LINT tool. It uses formal techniques to detect at least livelock, deadlock, accidental asynchronous circuits, and incomplete sensitivity lists. It's not clear if it also does normal Spyglass-style linting. It will tie up a formal engine license, so it's not clear if we have enough of those to add it to the check-in tests.
