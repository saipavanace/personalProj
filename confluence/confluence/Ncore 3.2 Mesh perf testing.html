The purpose of this study is to show how much performance could be achieved using NCore 3.2.

ARTG is being updated by software team, and in this test, we will apply only X-Y and Y-X algorithm. ARTG is not considering any performance input yet. 

Currently using software configuring switch which consists of splitter/merger, and considering to use sym_ibuf_switch.
