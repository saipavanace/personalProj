Sanjay Deshpande (Deactivated):

I suggest we change:

 

A Link is a unidirectional, point-to-point means of communication between a

pair of elements in an Interconnect. It is also referred to as a “Channel.” A

channel comprises of two unidirectional sets of wires (one set in each direction),

To:

 

A Link is a unidirectional, point-to-point means of communication between a

pair of elements in an Interconnect. It is also referred to as a “Channel.”

In its simplest form, a link comprises of a set of one or more parallel wires connecting the two elements and over which information

is transferred from one element to the other. The set of wires may also optionally include a subset of wires that the second element uses to signal back to the first element to control the

flow of information from it (link-level flow control).

 

A simplest bidirectional link has two sets of wires, one set of wires to transmit information from first element to the second, and second set of wires to transmit information from the second to the first.
