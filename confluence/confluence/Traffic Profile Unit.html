The Traffic Profile Unit provides a basic testbench to drive transactions to an AIU. It was initially designed with IOAIU in mind, but has been extended for CHIAIU. The features for the TPU is a minimal subset of the features described in ARM Traffic Profiles Specification to enable interesting random traffic.

Key Features of TPU

  • Specify transactions without regarding implementation details like bus widths or AIU type
  • Fixed sequence or randomized sequence of certain parameters of transactions
