How to administer the Jorani (French time management tool) instance (create user, update the counters, etc).

 Instructions

Instance is at the following address:

http://heretix8.arteris.com:8080

How to create a new user:

  1. Log in to the web interface as an administrator.

  2. Go to Admin → Create user. Fill in the form using the employee’s information from HR.
    The role will be “user”.
    If the manager doesn’t exist, it will need to be created.
    ”Contract” can be “manager” if the user is created only to validate an employee’s leaves (e.g. a manager with a US contract needs to validate a French employee’s leaves).
    “Position” doesn’t really matter and can just reflect the “Entity”.
    “Internal/Company Identifier” and “LDAP Path“ can be left blank.

Make sure to use the correct UNIX username (LDAP authentication) as login and that the e-mail address is correct as it is the user’s identity. They also need to be connected to the VPN to reach the host.

If the user has the error message “User account disabled” on the login page, they might have used their e-mail as their login instead of their UNIX username.

Once created, their counters need to be set to 0 at the date of their start day:

  1. Use HR → List of employees and double-click on a user to pop up the menu.

  2. Use the “Leave balance” menu to check a status of the counters for the given employee on the specified date of report.

  3. Use the “entitled days” to increase or decrease the leave counters (positive or negative numbers of days). Counters should logically be at 0 at the start day of their contract

“CP N-1”, as appear on the salary slips, are not pertinent in Jorani. They are counted as “CP”.
