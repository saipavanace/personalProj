    • Update UserParameters.xlsx, tag/data bank
    • Fix SelectBitsPrefix for SnoopFilters
    • addrDist label
    • Standardize parameter labels, change default of 'Coherent memory cache type' to 'Victim Cache'
  • CONC-2893 - CLONE - Resilience: Socket should only use cacheline aligned address bits to calculate incoming and outgoing parityRESOLVED
  • COND-785 - 'nSysCoherencyWindowPrefix' param value generated in achlParams is wrong RESOLVED

Resiliency is passing full system simulation on configs 5, 9, 11 and Skyrunner with faults injected on outputs of all Ncore instances.  Testing in progress still for protection module (place-holder blocks) for configs 5 and 9.  Generated RTL and placeholder signals look correct and connect to the full system top-level and to the fault checker.

Current known limitations in this release candidate are:

  • Customer TB now builds and is passing but not all checking is enabled.
  • SystemC / Perf is not functional.
  • Synthesis scripts and flow is untested and likely not working.
