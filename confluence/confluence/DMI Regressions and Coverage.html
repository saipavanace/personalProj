This is a live page for running DMI regressions / coverage.

Running DMI Regressions:

python run_regress.py +rtype=<ccov | mini | all | csr> +configs=<comma_separated_list_of_files> +sim=<questa | vcs>

Please specify all options. The defaults are set to mini, dmi_config1_v3 and vcs. Since Michael and Aniket are the only customers of this script, there is very little user-proofing / help for now.

Currently the +configs knob looks under /test_projects/achlProjects area. Please copy / link any apf you need to run into this area.

Example:

python run_regress.py +rtype=ccov +configs=dmi_config1_v3 +sim=questa

Common User Errors:

CONCERTO_TOP is not set.

For questa runs, unsetenv LD_LIBRARY_PATH and source /home/mentor/questa
