Example of Aria script for synthesis script generation

genDcSynScript({
topDesign:'ioaiu_top_a',
genScriptDir: 'dc_scripts_ioaiu_top_0',
cprInterfaceDirs: '../output/assets/interface',
rtlJson: '../output/debug/top.level.rtl.gw.csr.json',
rtlDir: '../output/rtl/design',
rtlWrapperDir: '../memories',
techNode: 'tsmc7',
topoMode: true,
checkOnly: false,
keepInteractiveSession: false,
bottomUpSynthesis: false,
incrementalOpt: true,
ungroupStartLevel: 2,
clockUncertainty: 0.15,
excludeVerilogModules: "eval getObjs({type:'instance'}).filter(o=>(o.module==='external_mem' && ((o.params.depth * o.params.width) >= 1024))).map(o=>o.params.verilogModule)",
hardMacroDbs: ['../memories/libraries_ssgn/*_lib.db'],
maxPaths: 10000
});
exit();

 

Notes

For zero-wire load synthesis, set the JSON parameter topoMode in the Aria script to false. Also, the JSON parameter clockUncertainty in the Aria script can be modified accordingly.
