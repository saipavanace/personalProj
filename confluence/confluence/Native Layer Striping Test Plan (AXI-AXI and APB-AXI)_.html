pamStrpGrp

MemoryMap0,1,20-2Array of integers representing the stripe group of the target
Maximum 3 stripe groups per ATUI.

nStrpTarg


1,2,4,8,16
Power of 2, upto 16

strpFunc

Fixed to 10,11

determines which address bits to use to find target

ValueAddress bits
0address[(N-1)+log2(num_targ):N] where N
1strpFuncHash represents address bits

strpFuncHash

MemoryMapArray of int
The array of integers represents what bits of address to use from LSB to MSB. Each entry represents a hash for an individual stripe group. 

strpSz

MemoryMapArray of int=interleavingSize (Network parameter)Array of integers representing the size of each stripe in the striping group.
StrpSz for given stripe group calculation:
Log2(StrpSz) < = {(address width of the initiator)-log2(#. of targets of the interleaving group)}

Eg: If a stripe group is of 4 targets and the total size allocated is 4 GB, the user can also specify the size of the region that every interleaved/striped target can access. The user can choose upto 1 GB of interleaving size in this case

strpMap

Fixed to 00,1,20Array of integers representing the address mapping function of the stripe group. (0 - None, 1 - Shift right to stripe bits. 2 - Move MSB to stripe bits)

TargetId


Array of int (Target Node IDs)
Node IDs of targets in a given stripe group

strpGrpRoute

MemoryMapArray of string
array of strings representing the route to be used by given stripe group

strpGrpMapBase

MemoryMapArray of stringNot implemented for R1array of strings representing the mapped base address of each entry.  "16'hF000" would be an example of an entry in the array.
