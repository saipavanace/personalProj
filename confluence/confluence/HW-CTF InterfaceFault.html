javascript parameters:

{ 
  "name": "a_string", 
  "params": 
     {
       wLateClk   : integer // Is either 1 or 0 indicating whether this signal is included,
     },
  "direction" : "master" or "slave", 
  "interface": "InterfaceFault"

}

Signals:

output mission_fault;                       // Signal indicating a mission fault.
output latent_fault;                        // Signal indicating a latent fault.
output cerr_over_thres_fault;               // Signal indicating the number of cerr went over the indicated threshold.
input  late_clk;                            // Late Clock input used to clock duplicate unit. This clock is synchronous to the functional clock. Can be excluded with wLateClk == 0

Description:

This interface defines the connection between a block and the Functional Safety Controller with the exception of the late clock. The late_clk will be driven from a synchronous clk to the functional clock used in the duplicate unit when unit duplication is enabled.
