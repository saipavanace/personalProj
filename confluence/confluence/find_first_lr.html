Descriptions

Find first (1 or 0 as specified) in an input bit vector beginning from left (most significant bit) to right (least significant bit).

Provide two outputs, both indicate the bit position of the 1 or 0 found.

One output is in onehot0 bit vector format.

Another output is in the binary encoded format.
