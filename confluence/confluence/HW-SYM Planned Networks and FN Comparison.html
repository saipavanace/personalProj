Functional Staging

Starting Point: All ATUIs and ATUTs configured as 256 bit wide AXI with following features turned off

  1. no narrows support
  2. no striping support

And with the following characteristics

  1. 256 outstanding

Run with three supported arbiter types in switches:

  1. arb_rr1
  2. arb_pri_rr1
  3. arb_fifo

Run with both packet types:

  1. parallel
  2. serial

Starting ATUI AXI config: HW-SYM Starting AXI ATUI config

Starting ATUT AXI config: HW-SYM Starting AXI ATUT Config

What do we want to vary?

  1. Turn on narrows.
  2. Turn on striping to targets 3,4,5,6 from all ATUIs
  3. Turn on config and interrupts
  4. Randomize ATUI, SW and ATUT data widths with 8,32,64,256, 1024, 2048 and auto insert adapters
  5. Turn on ROB
  6. Randomize number of outstanding to 2, 32, 256, 512
  7. Randomize number of PAM table entries to 7, 16, 128, 256, 512
  8. Randomize clocks to ATUI, SW and ATUT blocks and auto insert async adapters
  9. Replace some pipes and rate adapters with FIFOs and FIFO rate adapters.
  10. Run some of the FIFOs at half clock.
  11. Put in power domains and run UPF simulations.

Additional Networks

In addition to the Bronze design, Somewhere along the line we want to implement two larger designs. The difference would be if these additional networks are implemented in FN, the two designs no longer have to be mirror images of each other, just the most efficient design both sides can come up with that meet the design constraints.

The Silver Network

Below is the request side of the network. The response side is assumed to be a mirror image of the request. Since the networks will be auto routed, the routes may not be mirror images of each other.

The silver network is planned to be a 7x7 mesh with 49 initiators and 20 targets. The blue dotted lines adapter boxes above represent adapters or groups of adapters that may exists. It is desired that the following features be used by the sliver network:

  1. 2 GHz Frequency
  2. Multiple Power Domains
  3. Narrows
  4. Striping
  5. FIFO and FIFO rate adapters using half clock SRAMs
  6. Config and interrupt networks
  7. Firewalls
  8. dw_adapters
  9. ROB
