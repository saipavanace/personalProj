ATP Link Width Calculations:

For a given target data bus width dataWidth, the sum of all F_ and C_ fields is

Sum of all C_ fields

Plus the sum of all x(field)*wF_fieldMin such that n(F_data) = wF_data/wF_dataMin and x(F_data)= dataWidth/wF_dataMin and all n() and x() for all fields are equal and x() is a power of 2.

For Header Penalty, HP = 0 (Header Parallel)

ATP Link Width, wLink    = sum of all Base Packet Parameters for Header fields + sum of all F_ and C_ fields;

For Header Penalty, HP = 1 (Header Serial)

ATP Link Width, wLink = max(sum of all Base Packet Parameters for Header fields,(sum of all F_ and C_fields))

For Fixed Width, wLink = max(max(All H_ widths except H_addr)),(sum of all F_ and C_ fields))
