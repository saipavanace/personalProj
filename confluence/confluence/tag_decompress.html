Descriptions

tag_decompress is a module that reconstructs the address bits using the tag bits and the index bits.

tag_decompress performs the reverse operation of tag_compress and set_select. 

 
address -> [set_select], [tag_compress] -> [tag_decompress] -> address
 

tag_compress is used in the Common Cache Pipe (CCP) that is a generic cache controller IP module used by several projects.
