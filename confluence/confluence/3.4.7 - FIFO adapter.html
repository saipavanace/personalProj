The FIFO adapter is like a pipe adapter, except it uses SRAM for storage. Unlike the pipe adapter, there is no programmability of the internal structure or the ability to control pipeForward and pipeBackward.

Parameters

 

depth: The number of words in the SRAM. Can be 4 or greater.

hDepth: The number of headers that the FIFO can hold. If split = false, then the number of headers does not apply.

split: true or false.

The above parameters functional identically to the parameters defined in pipe adapter.
