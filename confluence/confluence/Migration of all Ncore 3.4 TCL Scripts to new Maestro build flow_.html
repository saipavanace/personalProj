How to Checkout and Run Simulations:

General instruction are at How to run NCore simulations:

Here are the steps to checkout Ncore3.4/develop and switch to particular maestro build (for bash user):

  • setup_concerto

  • source SOURCEME ncore3.4

  • cd ../hw-test-projects/

  • git config --add remote.origin.fetch +refs/heads/Ncore3.4/develop:refs/remotes/origin/Ncore3.4/develop

  • git fetch

  • git checkout Ncore3.4/develop

  • cd $WORK_TOP

  • export PATH="/engr/dev/releases/maestro/build/interleavingproject/maestro-product/stable/bin:/engr/dev/releases/maestro/build/interleavingproject/maestro-product/stable/maestro-server:${PATH}"

  • export MAESTRO_TCLLIB=/engr/dev/releases/maestro/build/interleavingproject/maestro-product/stable/maestro-client/tests/cases/test_tcl/tcllib
