JavaScript Parameters

{

“name”: a string that is used as a prefix for signals of the interface,
“direction”: “master” or “slave”,
“interface”: “InterfaceNodeId”,
“params”:
{
“wNodeId”: Width of node_id signal. A positive integer.
}

}

Signals

output [wNodeId-1:0] node_id; // output on a master interface, input on a slave interface

Description

This interface is used to provide a node identifier for a Symphony ATU. It is expected to be tied off by software (Maestro) to allow the possibility of having multiple ATU instances with different node IDs but can still share the same Verilog module definition.

The Node ID i.e. the value on the node_id signal is primarily used to drive the src_id signal on SMI request (on ATUIs) and response messages (on ATUTs) which helps identify the source of those messages.
