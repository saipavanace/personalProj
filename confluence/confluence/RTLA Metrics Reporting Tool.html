Tool: make_rtla_metrics


Description:

make_rtla_metrics

  • parses top level log files and reports.
  • collects results information and presents in a web browser.
  • Checks for design parameters such as ulvt percentage and cell count, and checks them against limits setup in the configuration file.
  • Checks for common errors and warnings and displays them within a web page.
  • It runs the bucketizer under the hood to create timing pages for all failing blocks


>make_rtla_metrics -h

Invocation:
make_syn_metrics <report dir>

Options:
-h/help
-nb (don't run the bucketizer)
-make_config_file (create the config file if you don't have one)
-v (returns version)


You first must create the config file:

> make_gen_wrapper_metrics.pl -make_config_file
