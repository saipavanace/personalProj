osci-lib repo contains all the TLM extensions for the various protocols (AXI, CHI, SMI, etc). 

Instructions on building

    • module load maestro
    • git clone http://gitlab.arteris.com/maestro-dev/packages/osci-lib.git
    • cd osci-lib
    • git checkout develop
    • git submodule init
    • git submodule update
    • mkdir build; cd build
    • cmake ..
    • make -j
