DCE0 DCE1 DCE2 DCE3
For DMI0: dmiDceConnectivity=[   1,   0,   1,   0]; //DMI0 connects to (and provides RB buffer entries for) DCE0, DCE2.
For DMI1: dmiDceConnectivity=[   0,   1,   0,   1]; //DMI1 connects to (and provides RB buffer entries for) DCE1, DCE3.
For DMI2: dmiDceConnectivity=[   1,   0,   1,   0]; //DMI2 connects to (and provides RB buffer entries for) DCE0, DCE2.
For DMI3: dmiDceConnectivity=[   0,   1,   0,   1]; //DMI3 connects to (and provides RB buffer entries for) DCE1, DCE3.

The number of the RB buffer entries per DCE is nDceRbCredits.
                             
DCE0 connects to DMI0, DMI2.
For DCE0: hexDceDmiRbOffset[0] = nDceRbCredits * 0 (DCE0 position in DMI0 is 0) = DMI0 RB Buffer Offset for DCE0
For DCE0: hexDceDmiRbOffset[1] = nDceRbCredits * 0 (DCE0 position in DMI2 is 0) = DMI2 RB Buffer Offset for DCE0

DCE1 connects to DMI1, DMI3.
For DCE1: hexDceDmiRbOffset[0] = nDceRbCredits * 0 (DCE1 position in DMI1 is 0) = DMI1 RB Buffer Offset for DCE1
For DCE1: hexDceDmiRbOffset[1] = nDceRbCredits * 0 (DCE1 position in DMI3 is 0) = DMI3 RB Buffer Offset for DCE1

DCE2 connects to DMI0, DMI2.
For DCE2: hexDceDmiRbOffset[0] = nDceRbCredits * 1 (DCE2 position in DMI0 is 1) = DMI0 RB Buffer Offset for DCE2
For DCE2: hexDceDmiRbOffset[1] = nDceRbCredits * 1 (DCE2 position in DMI2 is 1) = DMI2 RB Buffer Offset for DCE2

DCE3 connects to DMI1, DMI3.
For DCE3: hexDceDmiRbOffset[0] = nDceRbCredits * 1 (DCE3 position in DMI1 is 1) = DMI1 RB Buffer Offset for DCE3
For DCE3: hexDceDmiRbOffset[1] = nDceRbCredits * 1 (DCE3 position in DMI3 is 1) = DMI3 RB BUffer Offset for DCE3

Each DCE is given "hexDceConnectedDmiFUnitId" which is the array of FUnitId for the connected DMIs.

Each DCE has "nDceConnectedDmis" number of RB credit allocators.
Each RB credit allocator generates RBID number that ranges from 0 to nDceRbCredits-1.
The RBID that goes to DMI is RBID number + RBID offset.
