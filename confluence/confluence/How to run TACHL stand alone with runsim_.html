runsim -run tachl -e environment -tf tachl_file -p parameter_file -d output_directory

environment: Simulation Environment to pull search paths from
tachl_file: No path needed, just the file name.
parameter_file: The JSON parameter file. Full path is needed.
output_directory: Where the generated verilog is put. Full path is needed.
