javascript parameters:

{name: "a_string",
 params: 
  { "nVc" : positive integer, // number of virtual Channels
    "wBus" : positive integer 2 or greater, // Width of the bus signal 
    "wProt" : 0 or positive integer,  // width of side band forward protection signal  
    "wProt_b" : 0 or positive integer, // width of side band backward protection signal
    "wPress" : 0 or positive integer} // width of pressure signal
 direction: "slave", 
 interface: "InterfaceATP", 
 pktStyle: "serial", "parallel" or "fixed", 
 wData: positive integer //Must be a power of 2 multiple of wF_dataMin in Base Packet Parameters
 pktDef: 
  {packet: "a_string"}} // Name of the Link Packet Definition that will be generated dynamically by Software.

Signals:

output [nVc-1:0] valid; // Is the interface valid and which Virtual Channel. One or none hot.
input [nVc-1:0] ready; // Can the target accept a beat of data and on which Virtual Channel. Multiple or none can be asserted.
output first; // 1'b1 means first beat. 1'b0 means not first beat.
output last; // 1'b1 means last beat. 1'b0 means not last beat.
output [wBus-1:0] bus; // Data for the present beat of a packet.
output [wProt-1:0] sb_prot; //The side band forward protection on the interface for a beat. Can be a function of any of the outputs except for sb_prod. 
input [wProt_b-1:0] sb_prot_b; //The side band backward protection one the interface for a beat. Can be a function of any of the inputs except for sb_prot_b.
output [wPress-1:0] pressure; // The pressure being applied to the physical link.

Description:

InterfaceATP is the standard interface used to connect elements of a Symphony Interconnect.
