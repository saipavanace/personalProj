This is one of objects returned by the vlogGen( ) function found in the following path and file:
hw-lib/js/lib_utils.js

Description

The object is a collection of helper functions useful for working with objects generated by the Verilog expression generator factory functions.

Functions

Below are the list of functions that are available as part of this object:

cat ( exprObjs )

Return type: A vlogExpr object. The function takes in an array ( exprObjs param) of vlogExpr objects and returns a single vlogExpr object that is the result of concatenating all expression objects in the array.


bitwise ( op, exprObjs )

Return type: A vlogExpr object. The function that takes in an array ( exprObjs param) of vlogExpr objects and returns a single vlogExpr object that is the result of performing a bitwise binary operation specified by the op string param. Legal values for op: "|", "&", "^". Note that this is only meant for binary bitwise operations. For one's complement, use unaryOp ( ) function.


cmp ( op, lhs, rhs )

Return type: A vlogExpr object. Returns the result of comparing lhs vlogExpr object with rhs vlogExpr object. The type of comparison is specified via the op param which can take the following legal values: "==", ">", "<". ">=", "<=".


ternaryOp ( default, condition, branch )

Return type: A vlogExpr object. Returns the expression created using the conditional ternary operator "?". The expression will be of the form condition ? branch : default where condition, branch, and default are all vlogExpr objects.


oneHot2Bin ( signal )

Return type: A vlogExpr object. Returns the expression that encodes the vlogSignal parameter, signal, to a one-hot signal.


bin2OneHot ( signal )

Return type: A vlogExpr object. Returns the expression that encodes the vlogSignal parameter, signal ( assumed to be a one-hot encoded signal), to a binary-encoded signal.


thermo (signal )

Return type: A vlogExpr object. Returns the expression that encodes the vlogSignal parameter, signal, to a thermometer encoded signal i.e. each bit is the OR of all lower bits and the same bit in the input signal.
