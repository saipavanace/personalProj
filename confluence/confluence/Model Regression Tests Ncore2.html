So far, there are two versions of Apf for regression.

The original version is from Chirag’s directory; the other one is what we recently modified based on the original. Define the Apf from Chirag’s directory as Version 1 and modified Apf as Version 2.

 

Modifications in New Apf Files

Snoop Filters->Set selection bits->Primary selection bits

 

Coherent Agent Interfaces: Coherent Agent Interfaces->Resource Sizing->OTT entries, OTT

Coherent Agent Interfaces->CMD credit, DTR credit

 

Non-Coherent Bridge: Non-Coherent Bridges->Resource Sizing

Non-Coherent Bridges->Proxy Cache

 

DMI: Coherent Memory Cache
