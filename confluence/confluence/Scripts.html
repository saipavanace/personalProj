These scripts to assist in running tests are in perf-vc/tests/bin

gen_cfg.pl

This script is used to generate a set of jsons using scripts that call maestro on the queue. For more details, see Generate Configurations

run_list.pl

This is used for taking a list of tests, randomizing, and running on the queue. For more details, see Run a List of Tests

check_results.pl

This script summarizes the fails by configuration from previous runs of run_list.pl. It's primary use is for when reporting weekly results, Khaled would like to see the fails by configuration. For example, 

check_results.pl /scratch/meperfmod/jenkins/angelo_ncore3_*/angelo/build/run_list*
