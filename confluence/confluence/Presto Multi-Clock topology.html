To run the multi clock topology test in the current Symphony DV env.

1.Set up the env.

How to run Symphony simulations:

2.runsim -c top_multiclk_11x8 -l all //runs all the tests related to the config.

How to fix TACHL problems:

3. You can do a stand alone TACHL build by following the below instructions:

The design is located at:

$WORK_TOP/tb/top/jcodd_multi_width_cfg1_ni

Create a regression area: In that area type the command:

./hw/rtl/lib/bringup/run_multiwidth_mc

This will create the RTL and put it in a directory at the same level called:

multi_width_cfg1_mc

To do a quick lint check on the code type the following:

cd multi_width_cfg1_mc

verilator --lint-only ./gen_wrapper.v -I. --error-limit 0 -Wall -Wno-UNUSED
