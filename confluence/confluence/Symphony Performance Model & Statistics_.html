    How to run performance model simulations:

    STEP1 --> mkdir perf_model
    STEP2 --> cd perf_model
    STEP3 --> git clone git@gitlab.arteris.com:hardware/hw-sym-tlm.git 
    STEP4 --> cd hw-sym-tlm
    STEP4 --> npm i
    STEP5 --> npm run benes
    This will compile the benes network and run the simulation. At the end of simulation it will print following,

    Configuration Information:

    Simulation Statistics:

    Graphical Statistics:

    STEP6 --> cd ./build/benes4x4-5.js/c0/
    STEP7 --> ./sim -Dst  > temp.txt
    STEP8 --> python ../../../bin/post_stats.py --plot &
