hw-lib/js/lib_utils.js

Base Class

class VlogSignal

Composition

None

Description

This class is same as class VlogSignal except that each instance has a direction associated with it. Because a VlogPort is a type of VlogSignal, the same methods can be applied on both. This is not user-accessible and internally used to an convert interface to a VlogBundle objects so that the direction of signals in the interface are preserved. 

Properties

An instance of this class will have the following data properties:

  1. direction: A string that can have either one of the following values - "in", "out".
