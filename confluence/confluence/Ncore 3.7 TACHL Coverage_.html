TACHL Coverage Exclusions Syntax:

  • How to exclude TACHL code from TACHL coverage:

  • Rules to decide on Ncore exclusion labels are as follows:

    • We primarily use the label corresponding to the project. Eg: Ncore3.7 label would be ncore_3p7

    • If the designer deems the exclusion to be applicable to all variants of Ncore3 corresponding label would be ncore_3px

    • If the designer needs to add exclusions in common files/library components then those exclusions would be prefixed with the block name.

      • eg: if IOAIU needs to exclude certain lines within a library component for Ncore3.7, corresponding label would be ioaiu_3p7
