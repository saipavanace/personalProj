Directory Recall Issued:

A Victim Buffer entry, that is just saved to snoop filter for Victim Buffer Save On Write operation, is sent out as a Recall request by mistake, because the Victim Buffer entry is not cleared in time due to pipeline stall due to single-bit memory error in a previous Snoop Filter read operation. The Recall request should not be initiated when the pipeline is stalled.

This issue was found during DCE weekly regressions which mean test-case and checks was already present in DCE DV environment.
