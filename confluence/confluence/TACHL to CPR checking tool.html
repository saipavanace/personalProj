Create a regression area:

mkdir regress
cd regress
/engr/dev/tools/script/setup_sym -d master
source SOURCEME.tcsh

Create an area to run check and run check:

mkdir cpr_check
cd cpr_check

for hw-sym:
$WORK_TOP/../../hw-lib/js/check_cpr2src.js -w . -r $WORK_TOP/../.. -l $WORK_TOP/../../hw-sym -clo $WORK_TOP/../cpr/json/clo_file.txt

for hw-lib:
$WORK_TOP/../../hw-lib/js/check_cpr2src.js -w . -r $WORK_TOP/../.. -l $WORK_TOP/../../hw-lib -clo $WORK_TOP/../cpr/json/clo_file.txt

Running on Ncore blocks

Create an Ncore area and a place to run the script the run the script:

mkdir ncore
cd ncore
/engr/dev/tools/script/setup_concerto
source SOURCEME.csh
cd ..
mkdir check_cpr
cd check_cpr
$WORK_TOP/../hw-lib/js/check_cpr2src.js -w . -l $WORK_TOP/../hw-ncr -clo $WORK_TOP/../hw-ncr/cpr/json/clo_file.txt -r $WORK_TOP/.. -nc

The results will scroll by, but you can look at the results in the results.log file.
