Steps to populate Legato/Symphony repo part of Angelo

    • module load maestro

    • git clone git@gitlab.arteris.com:maestro-dev/packages/angelo.git

    • cd angelo/

    • module load maestro

    • git checkout integration

    • git submodule init

    • git submodule update --remote

    • setenv TEST_PRJ `pwd`/hw-test-projects

    • mkdir build

    • cd build

    • cmake -DCMAKE_BUILD_TYPE=Release  .. or (cmake -DCMAKE_BUILD_TYPE=Release -DLEGATO_TEST=dryrun_b.json ..)

    • cd hw-sym-tlm/build/dryrun_b.json ( or hw-sym-tlm/build/<testname>

    • make -j 12
