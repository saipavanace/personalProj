Flavours & tests

Test name formation

All available test name are formed using this formula

axi4_txn_adaptation_multi_beat_<Burst_type>_<Txn>_<Cross_feature(optional)>_<Narrow_feature(optional)>

Burst_type = incr, fixed, wrap
Txn = wr, rd, wr_rd
Cross_feature = unalign , mod, non_mod, exc

                      if(Burst_type==incr or Burst_type==wrap) valid Cross_feature strings = unalign , mod, non_mod, exc else valid Cross_feature strings =mod, non_mod, exc

                      if(string present==exc) test has exclusive txns else test has non-exclusive txns

Narrow_feature =

                      if(string present==narrow) test has narrow & non-narrow txns &  else test has non-narrow txns
