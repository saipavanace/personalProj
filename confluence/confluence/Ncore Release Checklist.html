The following is to be done for every release to customers:

  • Ensure that CSR registers that reference produce versions have been updated.
  • Update Open Source code list.
  • Review DV Coverage reports.
  • Review RAM Organization for each block.
  • Review Synthesis Warnings.
  • Review Customer TB Warnings.
  • Review documentation.
  • Review deferred bugs
  • Review PDF documents.
  • Review Error Messages
  • Review Ranges & Info boxes
  • Ensure generated reports are up to date.
  • Ensure generated .h and IP-XACT files are up to date.
  • Review version numbers of all libraries.
  • Install package review
    • Check for anything that shouldn't be there.
    • Ensure that the compiled versions of all libraries are included, no uncompiled versions.
