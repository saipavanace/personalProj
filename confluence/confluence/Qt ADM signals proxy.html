Object of class ADMSignalsProxy can be used for re translation of ADM signals to Qt objects.

Concepts:

  • ADMSignalsProxy subscribes to ADM signals
  • Filtering is used to narrow amount of signals
  • On ADM signal recieved ADMSignalsProxy emits one of tree pairs of qt signals : objectModified/objectRemoved/objectAdded with both ADMOwner pointer and object's hierarchical name

Filtering:

Filters can be splint into "pre connection" and "on emit". First group filters are set at connection time and actually signals are not emited if rules are not met. Second is more processing heavy as it it done on each signal emit.

Pre connection filters :

  • void setClassMaskFilter(std::vector<adm::AdmClassMask> &&maskVector);
    main filter that sets Object Class Mask for connection. Changes that are done for objects of this ClassType and it's children Types will emit signals. connection is created for every ClassMask in vector.

  • void setChangeTypeFilter(adm::AdmSignalBase::ChangeType type);
    Selects what changes to object emit signals. Possible values are:

    • ChangeType::MODIFIED

    • ChangeType::ADD
    • ChangeType::DEL
    • ChangeType::ALL

On emit filters:

  • void setNameFilter(std::vector<adm::UserPath> &&pathVector);
    Recieved object's user path is compared to vector of path's and qt signal is emited only if found. If vector is empty this is ignored.
  • void setFilterObject(adm::AdmOwner* obj);
    Recieved object is compared to filtering object. This also sets classMask filter.

Example use case:

  1. Create processing slot

    public slots:
	void onObjectModified(adm::AdmOwner *);

  2. Create instance and subscribe to desired signals

    ADMSignalsProxy *sigProxy = new ADMSignalsProxy("unique proxy name");
QObject::connect(sigProxy, SIGNAL(objectModified(adm::AdmOwner *)),
            this, SLOT(onObjectModified(adm::AdmOwner *)));

  3. Set filtering and create connections

    AdmOwner *obj;
sigProxy->setFilterObject(obj);
sigProxy->createSignals();
  4. onObjectModified will be called on changes of the object.

     

Resetting proxy:

    m_sigProxy->removeSignals();
    m_sigProxy->resetFilters();
