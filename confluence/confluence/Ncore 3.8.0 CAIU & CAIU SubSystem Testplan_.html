InitiatorID and TargetID change

The InitiatorId and TargetId are now a concatenation of {LinkId, ChipletId, FUnitId, PortId} which is an update from {FUnitId,PortId} for single die configuration The InitiatorId contains and propagates the same LinkId as the TargetId

InitiatorID and TargetID change incase of error logging

Right now for Error logging only FUnit_id is used. It might be updated to log FUnit_id and Chiplet_id.

Decide if a CmdReq needs to go to a DMI or a DCE in case of CMOs

Until Ncore 3.7 SnpAttr was desiding whether Req got to a DMI or a DCE. It will not be used anymore since GIU dose not have access to it.

For this reason, it was decided that the GIU will look at the CH attribute of the CmdReq which was a don’t care up until Ncore 3.7. If the CH attribute is set to 1, and the address hits a DMI region, then the transaction will be sent to DCE. If it is set to 0 it will be sent to DMI.

System event handling

Until 3.7 AIU was sending SysReq to all DVE and DCE available. In Die2Die configuration AIU will send SysReq to Local DVE only and each DVE in each chiplet will be responsible for the distribution of an event to the remote DVE and to the local event receivers

Unknown RequesterGlobalId

This happens when an AIU tries to attach but the Snoop filters are not configured yet.

Error Type: Uncorrectable, SoftwareProgConfig

Interrupt: Yes

Mission Fault: Yes

Logged information:
ErrType code: xC
ErrInfo: [15:6] GlobalRequesterId

Action: SysRsp CmStatus: {2’b01,ChipletId,1’b0,01 1}

Unknown Coherent Agent

This happens when an AIU sends a CmdReq but the Snoop filters are not configured yet.

Error Type: Uncorrectable, SoftwareProgConfig

Interrupt: Yes

Mission Fault: Yes

Logged information:
ErrType code: 0xC
ErrInfo: [3:0] 4’b0110: Unknown caching agent

Action: StrReq CmStatus: 8’b10000100. Do not continue sending snoops, do not update the directory. The native interface will propagate in the response as a decode error.

CXS_TX/RX timeout

Error Type: Uncorrectable, TimeoutErr

Interrupt: Yes

Mission Fault: Yes

Logged information:
ErrType code: 0xC

Action: The value for the timeout is 8 ms and if this timeout is reached it should stop the handshake.

CXS TX/RX Checktype error

Error Type: Uncorrectable, IntfCheckErr

Interrupt: Yes

Mission Fault: Yes

Logged information:
ErrType code: 0xD
ErrInfo: TX: 5’b10000
RX: 5’b10001

Action: Drop the transaction

Illegal start pointer

Error Type: Uncorrectable, TransErrInt

Interrupt: Yes

Mission Fault: Yes

Logged information:
ErrType code: 0x8

Action: Drop the flit.
