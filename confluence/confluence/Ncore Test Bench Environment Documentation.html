1.1 Doxygen Filter

Since Doxygen cannot recognize system verilog, in order to document the testbench, a filter needs to be applied which converts the constructs of system verilog into C++ which the doxygen can understand. Note that this is just a filter and not a code functionality conversion from System Verilog to C++

We use this filter for our documentation 

1.2 genDoc.py

This is a simple python script which grabs every file in the environment and applies the above filter to convert them into C++

This is the python script

3 Running Doxygen

  • Navigate to the "dv" directory after running a test
  • Copy the "doxygen_filter" file into this "dv" directory
  • Execute the following command > python genDoc.py 
  • cd to the cppClass directory that is created by the python script
  • copy the "config" file into this directory
  • Execute the following command > doxygen config
  • Open the generated index.html > firefox out/html/index.html
