Model Instrumentation

Modify the Symphony fabric model component to capture metrics during run time and store this metrics to a file using a pthread based logging mechanism based on G3Logger. Create a new module that is instantiated within or in parallel with each symphony switch element. This module accepts TLM packets and parses the contents to gather stats. This module attaches to arbiter ports to gather arbitration statistics.  These stats are gathered on a separate thread from the main execution and are sent to a logging module that buffers and emits the final file format.
