Proposal 4

The above picture can be extrapolated for Legato. Replace Trasport Layer, Control Layer, and Native Layer with all of Legatos functional blocks.

Each block will receive a protectionStyle and based on that style produce a protection interface. This protection interface will have a correctable and uncorrectable signal which the individual Legato block is responsible for driving. These protection interfaces are then accumulated with prot_muxs and driven to the fault_checker. The fault_checker will then convert these signals from the late and functional unit to mission/latent faults and report them upstream to the FSC.

Proposal 2


In this proposal Legato would have a single Functional Safety Controller that accumulates faults for every block.

Each block instantiates a Fault Checker which takes in the functional/late units outputs and un/correctable errors. It is also responsible for performing BIST on the functional safety signals.

Each protection interface will have its own un/correctable signals to the fault controller. Proposal 3

In this proposal Legato would have a single Functional Safety Controller that accumulates faults for every block.

Each block instantiates a Fault Checker which takes in the functional/late units outputs and un/correctable errors. It is also responsible for performing BIST on the functional safety signals.

Each protection will be grouped into two categories (has error correction/doesnt have error correction) as these two categories will have different mappings to un/correctable errors. Each group will accumulate the interfaces through a prot_mux. Then the mappings of errors from these outputs will be or'd together to a single un/correctable error interface at the top.
