javascript parameters:

{ "name": "a_string", 
  "direction" : "master" or "slave", 
  "blkClkGateOn" : true or false, // Block receiving clock interface 2nd level clock gate enable. 
  "parameter" : {"wTestEn" : 0 or 1,
                 "wClkEn" : 0 or 1, 
                 "wEdgeEn" : 0 or 1,
                 "wPreEdgeEn" : 0 or 1,
                 "wRetRst" : 0 or 1},
  "interface": "InterfaceCLK"
}

I/O:

output clk;
output reset_n;
if (wTestEn !== undefined && wTestEn === 1) output test_en;
if (wClkEn !== undefined && wClkEn === 1) output clk_en; 
if (wEdgeEn !== undefined && wEdgeEn === 1) output edge_en;
if (wPreEdgeEn !== undefined && wEdgeEn === 1) output pre_edge_en;
if (wRetRst !== undefined && wRetRst === 1) output retention_reset_n; 

Description:

Standard Clock Interface

The signals are used as follows:

 

pre_edge_en is used when having problems meeting timing with edge_en. If detected in an interface, logic will pass pre_edge_en through a flipflop to generate edge_en. If both signals exist, logic will give preference to pre_edge_en.
