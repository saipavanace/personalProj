Description

The class represents a lightweight abstraction of a contiguous range of bits / bit-positions on a bus. Every slice has a position and a size (width). An instance of this class provides the following properties:

  1. start. A positive integer or zero that specifies the most significant bit of the slice (Big endian).
  2. end. A positive integer or zero that specifies the least significant bit of the slice (Big endian).
  3. width. A positive integer or zero that specifies the width of the slice.

width is the width or size of the field in bits. The start and end mark the beginning and end bits of the field such that start > end whenever width > 0. This follows the common Verilog style of declaring bit vector sizes in big endian manner ( [ 5: 2 ] translates to start = 5, end = 2, width = 4 ). Even though an instance provides all three properties mentioned above, the implementation can be done by storing only two of the three properties with the third one implemented as a getter.

Interface (Methods)

All class methods described below are non-mutating (except the constructor). The class provides the following interface:

constructor ( { start, end, width } )

Creates an instance of Slice. The instance must be initialized with at least two of start, end, and width.


shift ( offset )

Returns a new Slice object that is a result of shifting the position of the Slice instance on which this is called by offset bits. The start or end of the returned Slice instance =  offset + start or end of the original Slice on which this is called.


switchEndian ( size )

Returns a new Slice object whose start and end positions reflect the change in endianness from "big" to "little" or vice-versa assuming the slice appears on a bus of width 'size'.


expand ( hold, size )

Returns a new Slice instance whose width is expanded by an amount size in bits, such that start of the returned slice remains the same as the original slice if hold === "start". Otherwise end of the returned slice remains the same as that of the original slice.


equal ( slice )   ( A == B )

Returns a boolean true if the parameter slice (assumed to be an instance of class Slice) is identical to the slice instance on which this is called in position and size. Otherwise returns false


intersect ( slice ) ( A intersection B )

Returns a new Slice instance that has all the bits that are common in both A and B. Returns undefined if not common bits were found.


exclude ( [ slice1, [ slice2, [ slice 3, ... sliceN ] ] ] ) (A - B - C - D... )

Returns an array of slices that consist of bits remaining after excluding bits in slices 1, 2, 3 ...N. This is equivalent to set-difference or relative complement in terms of sets (of bits).
