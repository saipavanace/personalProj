In Maestro, The following parameters have been added to support skid buffer size.

DMI

nCMDSkidBufSize

"title": "Total depth of the Skid Buffer",
            "min": 4,
            "default": 16,
            "max": 256,

nCMDSkidBufArb

"title": "Depth of skid buffer visible to arbitration",
            "min": 4,
            "default": 16,
            "enum": [
                4,
                8,
                16,
                32,
                64
            ],
            "max": 64,

nMrdSkidBufSize

"title": "Total depth of skid buffer for coherent DMI transactions",
            "min": 4,
            "default": 16,
            "max": 256,

nMrdSkidBufArb

"title": "Depth of skid buffer visible to arbitration for coherent DMI transactions",
            "min": 4,
            "default": 16,
            "enum": [
                4,
                8,
                16,
                32,
                64
            ],
            "max": 64,

using Native Tcl

set_attribute -object $dmi0 -name nCMDSkidBufSize    -value 8
set_attribute -object $dmi0 -name nCMDSkidBufArb     -value 4
set_attribute -object $dmi0 -name nMrdSkidBufSize    -value 8
set_attribute -object $dmi0 -name nMrdSkidBufArb     -value 4

using Tcllib

Dmi::set_nCMDSkidBufSize [Dmi::get_all_units] 8
Dmi::set_nCMDSkidBufArb  [Dmi::get_all_units] 4
Dmi::set_nMrdSkidBufSize [Dmi::get_all_units] 8
Dmi::set_nMrdSkidBufArb  [Dmi::get_all_units] 4

The following are deprecated and should not be used

CAIU

In CAIU, the following parameters are deprecated

nDceCmdCredits

nDmiCmdCredits

nDiiCmdCredits

In Native tcl , the following should be removed

set_attribute -object $caiu0 -name nDceCmdCredits        -value 8
set_attribute -object $caiu0 -name nDmiCmdCredits        -value 4
set_attribute -object $caiu0 -name nDiiCmdCredits        -value 4

In Tcllib, the following should be removed

Caiu::set_nDceCmdCredits [Caiu::get_all_units] 8
Caiu::set_nDmiCmdCredits [Caiu::get_all_units] 4
Caiu::set_nDiiCmdCredits [Caiu::get_all_units] 4

NCAIU

The following parameters are deprecated

nDceCmdCredits

nDmiCmdCredits

nDiiCmdCredits

In Native tcl, the following should be removed

set_attribute -object $ncaiu0 -name nDceCmdCredits   -value 8
set_attribute -object $ncaiu0 -name nDmiCmdCredits   -value 4

In Tcllib, the following should be removed

Ncaiu::set_nDceCmdCredits [Ncaiu::get_units {0}] 8
Ncaiu::set_nDmiCmdCredits [Ncaiu::get_units {0}] 4
Ncaiu::set_nDiiCmdCredits [Ncaiu::get_units {0}] 4
