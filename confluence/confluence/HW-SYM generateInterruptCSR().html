hw-lib/js/sym_csr_lib.js

Description

The function computes and returns the csr object for the interrupt block based on a number of parameters passed to it. Only the number of interrupts, numIn parameter must be passed. Other parameters are optional or have default values. This block is expected to be used by blocks that instantiate the interrupt block. The CSR returned by this function can then be merged with other registers or CSRs from other sub-blocks to create a CSR for the instantiating block. 

The function internally calls the HW-SYM genInterruptMap() function to calculate the interruptMap string as described in OBSOLETE-DELETE HW-SYM Symphony Block Common Characteristics using the parameters - level, tree, and srcBlockIDMap, when the parameter interruptMap is not passed. 

Usage

var csrInterrupt   = generateInterruptCSR( numIn, { addressWidth, id, width, baseAddress, tree, level, interruptMap, srcBlockIDMap });

Return Value

The csr object for the interrupt block that includes the INTVR and INTMR registers.

Parameters

  1. numIn: Type: number. Must be a positive integer. Indicates the number of input interrupts to the interrupt block. Cannot be greater than 32.
  2. id: (Optional) Type: number. Must be an integer. Sets the Block ID for the interrupt block and creates an IDR register with reset value = id. When it is undefined or negative, the IDR register is not created.
  3. width: (Optional) Type: number. Must be a positive integer and a power of 2. Defaults to 32 if parameter is not passed.
  4. baseAddress: (Optional) Type: number. Must be a positive integer.
  5. interruptMap: (Optional). Type: string. This is a string when defined or non-empty is appended to description of every register in the interrupt block and must follow the format mentioned in OBSOLETE-DELETE HW-SYM Symphony Block Common Characteristics under section titled "Interrupt Bit Description". If the this is undefined or empty, the below three parameters are used instead to calculate this description string.
  6. level : (Optional) Type: string. Same as the parameter described in HW-SYM genInterruptMap().
  7. tree: (Optional) Type: string. Legal values: "ERROR", "INFO".
  8. srcBlockIDMap: (Optional) Same as the parameter described in HW-SYM genInterruptMap().
