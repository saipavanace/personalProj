  • Burst and Narrow Conversions: Narrow bursts handled inside the native layer. Burst conversions will be supported inside the target CTL layer.
  • Byte Enables and Partial Accesses : The target CTL must return an error and know if it can support partial accesses.
  • Endieness Conversions: NDP is big endian and the datapath is little endian 
  • Transaction Alignments : Will be handled by burst aligner in target CTL.
  • User Data: Carried through SMI
  • Resiliency ans Interface Protection: Legato Functional Safety Proposal
  • Early Write Responses and Posted Writes: Can be enabled through enBufWrite
  • R5/R7 : Will be checked at initiator. Then the ATU will be protected through duplication. ATP will be protected by eend2end. Target will be protected by duplication. Protection will be recalculated and given on AXI.
