Description

This block is the wrapper used in Symphony to instantiate a one port SRAM (shared read and write) using the em_mem_external shared library element.

The em_mem_external block is the standard block used to create wrappers for SRAMs. The block's generated module name is in the form:

genInterfaceNew.name + 'em_mem_external'

So there must always be a genInterfaceNew object passed in, even if it contains 0 width signals.
