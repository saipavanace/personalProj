The NCore product software main executable is the 'conductor' bash script.  That script sets up the required system environment (libraries, etc) and then executes the javascript 'nw' program on 'my-xephyr.js'.  The 'my-xephyr.js' script sets up the X-Windows environment and then executes the javascript 'nw' program on 'cpanel'.  The files for 'conductor', 'my-xephyr.js', and 'cpanel' are contained within the 'client' repository.  The other NCore packages are all used as components within 'cpanel'.

Achl

The achl package contains the ACHL (Arteris Configurable Hardware Language) compiler code.

  • To view achl code online: https://stash.arteris.com/projects/CON/repos/achl/browse

  • To 'git clone' a local achl repository:

    % git clone ssh://git@stash.arteris.com:7999/con/achl.git

Client

The client package hosts the 'conductor' UI wrapper container. The container runs Xephyr (an X11 server) and uses the Conductor Window Manager (cwm) as the key components.  The code in the cpanel subdirectory is the main code driving NOC configuration. 

  • To view client code online: https://stash.arteris.com/projects/CON/repos/client/browse

  • To 'git clone' a local client repository:

    % git clone ssh://git@stash.arteris.com:7999/con/client.git

Decaf

The decaf (Design Exchange, Concerto And FlexNoC) package is a library for reading and modifying FlexNoC pdd files. It is used to add Concerto specific sockets and connections to a FlexNoC pdd file, as well as querying the pdd file for clocking information needed by Concerto to generate RTL code consistent with that produced by FlexNoC.

  • To view decaf code online: https://stash.arteris.com/projects/CON/repos/decaf/browse

  • To 'git clone' a local decaf repository:

    % git clone ssh://git@stash.arteris.com:7999/con/decaf.git
