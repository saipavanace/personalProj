This function returns the number of beats needed to grab all the fields asked for from the packet, a list of signals to based on the fields asked for and the output for each field asked for from HW-CTF getFieldPosAndBeatInLinkPacket. The packet description can be a base packet description or link packet description.

It is called as follows:

var return_data =  getFieldsFromPkt(link_width,link_packet_def,field_array)

link_width : width of the link in bits

link_packet_def : packetDesc

field_array: An array of text strings used to match against fieldNames. This string is matched against the beginning of the fieldNames in the packet description. In other words, if field_name is 8 characters long, and the fieldName in the packet is 12 characters long, field_name is compared against the first 8 characters of that 12 character fieldName and if those 8 characters match, it is considered a match.

It returns a variable with the following format:

return_data = {
  "beats" : non negative integer,
  "signals" : {interface description},
  "fieldPos" : [] //an array of data returned by HW-CTF getFieldPosAndBeatInLinkPacket, one per entry in field_array.

}
