NOTES:

  1. CTL Interface A: Interface definitions for one type of NoC (let's say A).
  2. CTL Interface B: Interface definitions for NoC B
  3. SHIM: Does the conversion of Interface definitions A<->B. By default
    1. Pads signals
    2. Truncates Signals
    3. Drives signals with mapped value.
  4. Mapped Value: Can be a constant or another signal. 
          1. For Example: If User bits are not present in interface A and present in interface B, selected fields of Address bits in A can be mapped into User bits of B.
  5. ROB: Re-Order Buffer

CONFIGS:

  1. Single VC 1x1
  2. Single VC 1x4
  3. Multiple VC 1x1
  4. Multi VC 1x4
