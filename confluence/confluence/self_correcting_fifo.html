Descriptions

Self-correcting FIFO module with optional support for error detection/correction.

The FIFO push_data carries data bits and optional error encoding bits. The FIFO push_data is transferred with push_valid=1 and push_ready=1.

The FIFO pop_data carries data bits only, and is accompanied by the error status signals pop_cancel and uncorrectable_error.

The FIFO pop_data is transferred when pop_valid=1, pop_ready=1 and pop_cancel=0.

The FIFO pop_cancel is asserted to indicate that the pop_data contains an correctable error, and will be asserted in the next clock cycle to indicate that the pop_data contains corrected data.
