Issue: When we called 'uvm_fatal () macro from testbench it its kills the simulation right away. This way we do not get any summary or report about the simulation.

Solution: Basically add the following function...

virtual function void pre_abort();

    // call what needs to run at UVM_FATAL

    // for instance, I want to run report_phase

    this.report_phase(null);

endfunction
