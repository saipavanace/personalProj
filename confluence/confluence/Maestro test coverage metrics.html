Building tests coverage metrics locally.

Build and generate all tests coverage.

  1. Build Maestro using using 

    make build_cov

    (see target here)

  2. Run tests and generate rapport using

    make test_only_cov

Build and generate GUI only tests coverage.

  1. Build Maestro using using 

    make build_cov

  2. Run gui tests using:

    # cleanup all previous coverage data (delete all *.gcda files)
find -type f -name '*.gcda' | xargs rm

# run tests (will regenerate .gcda files)
cd build_cov && ctest -R “tst_gui*”

  3. generate GUI coverage data and rapport:

    mkdir -p coverage
cd coverage; ../scripts/metrics/gui_coverage.sh
Firefox coverage/gui/index.html

    (see script file here)
