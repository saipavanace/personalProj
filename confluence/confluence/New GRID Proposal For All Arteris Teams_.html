Licenses Management:

  1. Support all type of simulator licenses (Synopsys, Siemens & Cadence).

  2. Support all type of VIPs licenses (AXI, APB, CHI-E, etc.).

  3. Allocation of licenses based on each team pre-allocated share.

  4. Fair share allocation within each team members.

  5. Able to arbitrate based on HARD and/or SOFT limits.

  6. Support all Arteris teams (Ncore, FlexNoC, Coda-Cache, CSR-Compiler, etc.)

Physical Server / Slots Management:

  1. Can be physically located anywhere in the world.

  2. Each slot must be consistent of exactly same OS and tool mounts.

  3. Some slots must configure with high RAM footprint to hand memory intensive simulations.

  4. Number of slots must be equal or greater than max number of all simulator licenses.
