parameters

java_script:

{
	"data_width" : any positive integer, 
        "pipeErrSb" : boolean, //when pipeEn == true, true cause the outputs err_sb to be piped by one clock 
        "pipeErrDb" : boolean, //when pipeEn == true, true cause the outputs err_db to be piped by one clock 
        "pipeSyndrome" : boolean // when pipeEn == true, causes the output syndrome to be piped by one clock
}

none.

I/O

input din[width-1:0];

input ecc[ecc_width-1:0];                        // The original ecc bits

output syndrome[ecc_width-1:0];           // The calculated syndrome, ecc_width calculated from width using secdedFunctions.getEccWidth

output err_sb;                                       // single bit error

output err_db;                                       // double bit error

if (pipeErrSb | pipeErrDb | pipeSyndrome) {

  input clk;

  input reset_n;

}

Description

Takes in data and ecc and outputs the syndrome, single bit error and double bit error.

The code uses a Modified Hamming Code for the SECED encoding.

When pipeErrSb is set to true, the output signals err_sb is piped by one clock.

When pipeErrDb is set to true, the output signals err_db is piped by one clock.

When pipeEn == true and pipeSyndrome is set to true, the output signal syndrome is piped by one clock.
