This is a JS class that can be found in the following path and file:
hw-lib/js/lib_utils.js

Base Class

None

Composition

class VlogSignal

Description

This class is used to represent a Verilog array data-structure that can be indexed into in-order to access each element of the array. Each element of the array in turn is an instance of class VlogSignal.

Properties

An instance of this class will have the following data properties:

  1. length: Length of the array  = number of signal slice elements in the array = number of chunks of the whole signal.

Interface (Methods)

The class provides the following interface methods:

constructor ( width, name, length )

Creates an instance of VlogArray. Parameter width is the width of each signal in the array. name is the name of the Verilog array. length is the length of the array.

idx ( i )

Return type: VlogSignal. Returns the ith signal in the array. Because there is no support for operator-overloading in JS, this acts as a substitute for indexing operator, [ ]. 

entries ( )

Return type: iterator. At each step the iterator fetches (value) an array of two elements: [ i, signal ], where i is the index and signal is the ith element ( class VlogSignal instance) in the array. The return type is modeled after the built-in Array method entries().

declare( type )

Return type: string. This returns the declaration of the Verilog array of the specified type ("wire" or "reg").

Default Iterator

Any instance of this class has a default iterator that fetches (value) each signal in the array. In order to get an array from an iterator, one could use Array.from( ) or array destructuring instead.

Examples

const register                 = VlogArray ( 8, "reg", 4 );
\=register.declare("wire")=\                                  // prints "wire [7: 0] reg [ 0 : 3 ];"
register.idx( 2 )                                                      // Returns expression reg [ 2 ]
register.idx( 2 ).lower(4)                                        // Returns expression reg [ 2 ] [ 3 : 0 ];

const registerSignals     = Array.from( register );  // Returns an array of signals [ reg[0], reg[1], reg[2], reg[3] ]
OR
const registerSignals     = [ ...register ];                // Returns an array of signals [ reg[0], reg[1], reg[2], reg[3] ]

OR
// Use iterator directly to parse with a for-of loop
for ( const byte of register ) {
   // Perform operations or assign on each byte of the 32-bit "register_bits" signal
}
