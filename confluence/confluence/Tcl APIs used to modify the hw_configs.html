Config_design 

Power parameters

Create power

Power::create <name of power region> <name for pwrdom> <voltage> <gating>

Enable PMA with switchable power 

Example: Power::create "pwrreg" "pwrdom" "1.0V" "dynamic" 

If Power gating is "dynamic", the clock must compulsorily be set to external.

Clock::set_external_gating <clock> true

Enable always_on without external power 

Example: Power::create "pwrreg" "pwrdom" "1.0V" "always_on"

If Power gating is "always_on", the clock gating can be either external or not.

Clock::set_external_gating <clock> true/false

Access power region

Power::get_region

Access power domain by name

Power::get_domain_by_name 

Example: 

Power::create "pwrreg" "pwrdom1" "1.0V" "always_on"

set pwrdomain1 [Power::get_domain_by_name "pwrdom1"]

Clock::set_power_domain    $clock_22_GHz $pwrdomain1

Clock parameters

Create clock

Clock::create <name for clk_region> <name for clkdomain> <name for clksub> <frequency>

Example: Clock::create "clkreg_a" "clkdom_a" "clksubdom_a" "2.2GHz"

Enable PMA

Clock::set_external_gating <clock> <true/false>

Example: Clock::set_external_gating $clock_16_GHz true


Enable unit clock gating

Clock::set_unit_gating <clock> <true/false>

Example: Clock::set_external_gating $clock_16_GHz true


Access individual clock by frequency 

Clock::get_by_freq <frequency given while creating>

Example: Clock::get_by_freq "2.2GHz"

Socket parameters

Create socket

Socket::create -name <name> -protocol <protocol> -function <INITIATOR/MEMORY/PERIPHERAL> -role <SLAVE/MASTER> -wdata <Datawidth> -waddr <Addr width> -clock <clk>

Example:Socket::create -name "caiu0" -protocol CHI-B -function INITIATOR -role SLAVE -wdata 256 -waddr 50 -clock $clock_22_GHz

Example:Socket::create -name "dmi0" -protocol AXI4 -function MEMORY -role MASTER -wdata 256 -waddr 50 -clock $clock_16_GHz

Access individual sockets by name

Socket::get_by_name < names of the sockets>

Example: Socket::get_by_name [list "dmi0" "dmi1" "dmi2" "dmi3"]
                Socket::get_by_name [list "ncaiu0" "ncaiu1" "ncaiu2" "ncaiu3"]
                Socket::get_by_name [list "caiu0" "caiu1" "caiu2" "caiu3"]
                Socket::get_by_name [list "dii0" "dii1" "dii2" "dii3"]

Set parameters on socket

Socket::set_eAc     <list_of_sockets> <value>    // To enable DVM
Example: Socket::set_eAc [Socket::get_by_name [list "ncaiu2" "ncaiu3" "ncaiu4"]] 1
