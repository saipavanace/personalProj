Purpose:  quickstart 

 

  1. choose a tree <my_tree> created with latest_concerto, from which to 

    1. cd <my_tree>

    2. ln -s concerto hw-ncr

  2. choose a tcl dir at <my_tcl_dir>

    1. ( choose a dir of form:  .../test_projects/ncore_v3.0_configs/<my_unit>/<my_config> )

    2. cd <my_tcl_dir>

    3. ln -s <your top tcl> dryrun.tcl


  3. update your test_projects

     

  4. on centos 7

    1. ssh -X gr6-3

  5. cleanup: run   ./run_maestro.sh -c

  6. -- to generate jsons with latest known good sources

    1. (see cmds at top of run_maestro.sh for latest known good)

    2. <my_tcl_dir>/../../sa_dryrun/run_maestro.sh \

      -mc /home/dant/arteris/maestro/build/bin/maestro \

      -rp /engr/dev/releases/utils/dep/m.ncore.0.3.0

  7. -- else to to generate jsons with local environment
    1. <my_tcl_dir>/../../sa_dryrun/run_maestro.sh \
      -mc /home/dant/arteris/maestro/build/bin/maestro \
      -rp <my_tree>

  8. To generate rtl, add the option -g   <-- RTL quality WIP
    1. all options read run_maestro.sh

  9. dv and rtl jsons and generated rtl are now in ./output
