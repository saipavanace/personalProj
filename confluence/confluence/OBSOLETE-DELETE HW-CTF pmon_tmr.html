module_name:

pmon_stats

JavaScript parameters:

width: This is a positive integer that specifies the width of the timer or counter. This needs to match the width of the val_in and val_out signals of the cfgValInterface of the timer.

Interfaces:

clkInterface: A slave interface of type InterfaceCLK.

cfgValInterface: A slave interface with the following signals and their widths:
{
  val_in: width,
  val_out: -width,
  val_wr: -1
}

cfgCtrlInterface: A slave interface with the following signals and widths:
{
  start_in: 1,
  stop_in: 1,
  reset_in: 1
}

cfgEnModeInterface: A slave interface with the following signals and widths:
{
  en_in: 1,
  inc_mode_in: 2,
  ct_in: 1
}

cfgLmtInterface: A slave interface with the following signals and widths:

{
  limit_in: width,
}

evtTmrInterface: A slave interface with the following signals and widths:
{
  start: 1,
  stop: 1,
  reset: 1,
  count: 1,
  tmr: -1 or 0 (optional) 

}

Block description:

The PMON timer block
