Release notes for Beta3 September 23rd release.

RTL features enabled in Ncore3.4 July Beta3 release:

  • Proxy cache NC-AIU interleaving only at 64B byte boundaries

  •  Software credit management and skid buffer

  •  No interleaving on NCAIU data responses

  •  Proxy cache support for owner transfer

  •  Proxy cache support for sharer promotion

  •  NCAIU with proxy cache non-coherent access support

  •  Support for security and coherency based on GPRS

  •  Performance counter (bandwidth, latency)

 Timing improvements:

  • Worked on improving timing paths in DCE, DMI & IOAIU.

 

Errata & Release notes:

  • There could be functional issues with sharer promotion when used in ACE configuration.

  • There could be functional issues when allocating non-coherent data in proxy cache.

    • Workaround: Avoid allocating non-coherent data in proxy cache.

Functional issues:

  • CONC-10514: IO: assert_dtr - r_message_id contained in the dtr_req_rx does not match an entry which has wfdtr asserted

  • CONC-10509: Noncoh_proxy: potential livelock between wakes and evictions

  • CONC-10502: OVL_FATAL : ASSERT_NEVER tb_top.dut.ncaiu0.ioaiu_core_wrapper.ioaiu_core0.ioaiu_control.assert_dtr.ovl_error_t
