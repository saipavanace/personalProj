Create a symphony area:

mkdir symphony
cd symphony
/engr/dev/tools/script/setup_sym
source SOURCEME

Go somewhere, anywhere and run the script:

mkdir constraints
mkdir cover_files
$WORK_TOP/../../hw-lib/js/createTypeDoc.js -o doc.txt -t $WORK_TOP/../type/Ncore/rtlprimary/symphony_elements -t $WORK_TOP/../type/Ncore/interfaces -t $WORK_TOP/../type/Ncore/rtlprimary/sym_packet -co ./cover_files -c ./constraints -wl

And the document has been created in outputted to doc.txt and the constraint files will be written out to ./constraints.

For a usage type 

$WORK_TOP/../../hw-lib/js/createTypeDoc.js -h
ERROR - /scratch/jcodd/regress/hw-lib/js/createTypeDoc.js: Missing type directores - there must be at least one.
ERROR - /scratch/jcodd/regress/hw-lib/js/createTypeDoc.js: Missing Document files
/scratch/jcodd/regress/hw-lib/js/createTypeDoc.js Usage -t typeDir -o outDoc <-t <more typeDirs -co coverDir -c constDir -wl -po -pm -cc -h>

typeDir: Required. Path to type directory, must be atl least once
outDoc: Required. Path to outDoc file
covDir: Optional. Path to covDir, can be more than one.
constDir: Optional. Directory to write constraints and white list.
-wl: Optional. Print out only whitelist constraints
-po: Optional. Print out paths only, then exit
-pm: Optional. Causes only postMap parameters to make it to constraints
-cc: Optional. Create Cover Files. If set creates a set of coverfiles
-ue: Optional. If -cc and -ue, make all user editable parameters have whitelist=true.
-h: Optional. Prints out this message

Before you can create constraints for the type files, you need cover files to edit to add constraints to. You can create these files with the -cc option and they will be written into the first coverDir you pass in.

Once you have a the cover file for a type file, you modify it by adding a parameterCoverage to it:

Before Edit

{
  "type": "atui_apb",
  "baseType": null,
  "members": {
  "interfaces": {
    "kind": "Object"
  },
  "timeoutErrCount": {
    "kind": "Variable",
  }
}

After Edit

{
  "type": "atui_apb",
  "baseType": null,
  "members": {
  "interfaces": {
    "kind": "Object"
  },
  "timeoutErrCount": {
    "kind": "Variable",
    "parameterCoverage": {
      "whitelist" : true,

        0,

}
