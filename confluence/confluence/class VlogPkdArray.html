This is a JS class that can be found in the following path and file:
hw-lib/js/lib_utils.js

Base Class

class VlogSignal

Composition

class PkdArray

Description

This class is used to represent a Verilog signal as a packed array i.e. each element of the array corresponds to one of n equal-width chunks (signal slices) of the signal and can be accessed via an index.

A chunk is an instance of VlogNet returned when a slice operation is performed on the packed array signal. Each chunk is generated dynamically during a method call that returns a chunk, instead of creating and storing them during the constructor call to save memory.

Properties

An instance of this class will have the following data properties:

  1. length: Length of the array  = number of signal slice elements in the array = number of chunks of the whole signal.

Interface (Methods)

The class provides the following interface methods:

constructor ( size, name, length )

Creates an instance of VlogPkdArray. Parameter size is the size of each chunk. name is the name of the whole signal. length is the length of the array or number of chunks. Because a VlogPkdArray is a VlogSignal, it has a width = size * length.

idx ( i )

Return type: VlogNet. Returns the ith chunk of the signal. Because there is no support for operator-overloading in JS, this acts as a substitute for indexing operator, [ ]. 

entries ( )

Return type: iterator. At each step the iterator fetches (value) an array of two elements: [ i, chunk ], where i is the index and chunk is the ith chunk of the packet signal that is sliced. The return type is modeled after the built-in Array method entries().

Default Iterator

Any instance of this class has a default iterator that fetches (value) each chunk of the signal starting from index 0 till the last index. In order to get an array from an iterator, one could use Array.from( ) or array destructuring instead.

Examples

const register                 = VlogPkdArray ( 8, "register_bits", 4 );
register.idx( 2 )                                                      // Returns register_bits [ 23 : 16 ]

const registerChunks     = Array.from( register );  // Returns an array of bit-vectors/net chunks [ register_bits[7:0], register_bits[15:8], register_bits[23:16], register_bits[31:24] ]
OR
const registerChunks     = [ ...register ];                // Returns an array of bit-vectors/nets chunks [ register_bits[7:0], register_bits[15:8], register_bits[23:16], register_bits[31:24] ]

OR
// Use iterator directly to parse with a for-of loop
for ( const byte of register ) {
   // Perform operations or assign on each byte of the 32-bit "register_bits" signal
}

Conversion to a packed signal

If an unpacked signal already exists then it can be converted to a packed signal using the function vlogSignal2PkdArray( length, name) in order to break it into length number of chunks.
