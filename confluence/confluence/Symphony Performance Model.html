Repositories

hardware/symphony is the main Repo for the Performance model of Symphony.  This repo's purpose is to hold the top level symphony model along with all supported configurations and tests.

Repository Structure

The directory structure of this repo is as follows:

Creating a working copy (single step)

git clone --recursive git@gitlab.arteris.com:hardware/symphony.git
cd symphony

Creating a working copy (step by step)

git clone git@gitlab.arteris.com:hardware/symphony.git
cd symphony
git submodule init
git submodule update --recursive --init

Updating a working copy aka pull

cd angelo
git pull origin integration
git submodule sync
git submodule update --recursive

Build Instructions (incl. running regression test)

cd symphony
module purge
module load dev/angelo
setenv TEST_PRJ `pwd`/hw-test
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j
make test
