Circuit Effect Coefficients

  • Single number per run
  • Input Rate - 48%
    • Q - Conflicts per output port per cycle
      • Q = ∑( number of output ports with conflict over entire run) / (num of output ports per switch * num clks)
    • Q1 - Pending requests per output port per cycle
      • Q1 = ∑( number of reqs pending per output port per cycle) / (num of output ports per switch * num clks) 



Finite Buffering at Target

For the results generated below, the target was re-designed to accommodate a programmable finite number of requests. Once this buffer/queue fills up, the NoC sees backpressure, which results in reduced throughput.
