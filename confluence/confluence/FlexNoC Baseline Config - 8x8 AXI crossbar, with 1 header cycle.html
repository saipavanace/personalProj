AXI1283244000

 

In this configuration the AXI NIUs:

  • do not support narrow bursts, but support single narrow accesses
  • max wrapping burst size = 64 bytes; the wrapping burst alignments are 16B
  • target NIUs do not support exclusive accesses, and there are no exclusive monitors
  • Will never assert DECERR (i.e. all errors coming back will be denoted as SLVERR)
  • assume initiator IP's AXI interface has the same capabilities as the target IP's AXI interface - i.e. no conversions need to be done at the NIU

Synthesis conditions

  • Library: tcbn16ffplusglbwp16p90ssgnp0p72v125c

  • DC version : L-2016.03-SP2

  • IO budget: 30% of the clock period for outside, 70% for the NoC
  • Clock uncertainty = 0
  • Everything flattened from the top
  • All clock gating enabled (synthesis level + FlexNoC unit level)
  • zero wire load
  • 2-step compile: 1. map to generic cells, 2. optimizations
