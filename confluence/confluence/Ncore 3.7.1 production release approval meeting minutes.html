Notes

Hi all,

Thanks for attending the Ncore 3.7 production release signoff meeting held March-14th from 10AM to 11AM PDT to review the product status and confirm there is no gating item including the QA for the product to be released to the customers.

The supporting data are available in the Ncore 3.7.0 production release checklist

See below the notes I have taken and the action items captured during the meeting.

Product scope:

Among the approved PRDs and new features for Ncore 3.7 project, the 3 items below have been pushed out to Ncore 3.7.1:

  • NCR37-PRD-331: Ordered Write Observation (OWO) support

  • NCOR-605: Add CheckType property on AXI/ ACE/ ACE_Lite IO-AIU interface

  • NCR37-PRD-328: FMEDA export for custom automotive configuration

The change of the scope has been approved by the Ncore 3 PCT.

RTL signoff status:

Timing results

Few units have timing paths not met with worst negative slack WNS < 100ps except one unit with a WNS = 106ps.

The STA is run with RTLA and TSMC 5nm technology node.

The worst corner is "ssgnp_0p675v_m40c_cworst_CCworst_T.cworst_CCworst_T" and the STA results are reported for that corner.

Fewer timing violations are reported with the "ssgnp_0p675v_125c_cworst_CCworst_T.rcworst_CCworst_T" corner and the timing is clean with the Typical corner. 

AI #1: Verify if at least one configuration uses a multi-cycle SRAM and timing report exist

Answer: DMI units are by default implemented with multi-cycle SRAMs when supported and timing is validated with them in Ncore 3.7 using "hw_cfg_10_lite" configuration.

AI #2: Verify timing with IO-AIUp for Ncore 3.7.1 => own = Eric H.

Lint results:

There are known violations with severity "error" that are reported by the Synopsys tool and warnings by the Cadence tool.

Many JIRA tickets related to Lint issues have been resolved for Ncore 3.7 but some tickets are still open until the right timing window is available to fix the code without impacting a product release.
In the meantime, a custom Lint waiver report will be provided to customers that request it. 

Open issues waived for that release:

The link to the list of issues is provided in the checklist. 10 JIRA tickets are new versus Ncore 3.6.4.p3 and 11 JIRA tickets are Lint issues.
