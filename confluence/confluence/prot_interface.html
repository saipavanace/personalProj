}

Description:

 

When the mode is forward, the block takes the protection scheme and applies it on the inputs of the inInterface and generates a prot signal which is an output in the outInterfaces. The inInterface need not have a prot signal in, but the outInterface must and it's width must match the width needed by the protectionScheme.

When the mode is backward, the block takes the protections scheme and applies it on the inputs of the outInterface and generates a prot_b signal which is an output in the inInterface. The outInterface need not have a prot_b signal in, but the inInterface must and its width must match the width needed by the protectionScheme.
