GATE AND SPACE DELAY

  • There is not a single answer, it is a compromise between Area and Frequency.
  • Space delay and Gate delay affect different part of the design.
  • Space delay is relatively straight forward, it doesn't affect the NIU delay.
  • Flow:
    1. Begin to use the Suggested Gate delay for the Technology
    2. Use suggested value by auto-pipe if failed
    3. If failed modify Topologogy
    4. Iterate between these steps
    5. If Design requires a backup solution ( advance capability already added) is also available in 2.2 (but not documented) which allows more flexibility using Space Delay.
  • Provide training to the AEs on this flow
