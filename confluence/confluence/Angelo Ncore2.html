angelo ncore2 repo contains all the source code for Ncore Unit performance models and required adapters to interface with external wrappers. 

Dependencies for this repo are osci-lib, legato, tempo.

Instructions on building this repo for PA integration:

  • unsetenv LD_LIBRARY_PATH
  • module load maestro

  • module load snps/pa/ultra/O-2018.09-SP1-1u3/gcc-5.2.0-64

  • git clone --recursive ssh://git@stash.arteris.com:7999/maes/angelo.git
  • cd angelo
  • mkdir build; cd build
  • cmake -DCMAKE_BUILD_TYPE=Release ../
  • make -j

Instructions on building this repo for OSCI stand-alone and run the regression tests:

  • module load dev/angelo/nc2_osci231_gcc63
  • git clone --recursive ssh://git@stash.arteris.com:7999/maes/angelo.git
  • cd angelo
  • mkdir build; cd build
  • cmake -DCMAKE_BUILD_TYPE=Release -DHAS_PCT=OFF -DHAS_SYSTEMC=ON ../
  • For debug build
    • cmake -DHAS_PCT=OFF -DHAS_SYSTEMC=ON -DCMAKE_BUILD_TYPE=Debug -DEN_DEBUG=ON ../
  • make -j
  • ( cd .. ; ./scripts/run-osci-regr )

Instructions on pulling changes from central repo:

  • cd angelo
  • git pull
  • git submodule sync
  • git submodule update --recursive
