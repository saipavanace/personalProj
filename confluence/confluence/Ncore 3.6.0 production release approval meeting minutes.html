RTL signoff status:

Lint results:

List of waivers is provided.

RTL open JIRA tickets:

One open RTL bug has been analyzed:

  1. CONC-13412        [DCE] dmi not present error cannot be generated

The root cause of the failure is that the DCE is not connected to one of the DMI units but there is a transaction with the address mapped to this DMI.

This issue is not a blocker for the release, it can be kept internal and AEs can find that issue for their reference.

Timing analysis:

DC-NXT run completed for NXP and HW-Config10-Lite configurations and the signoff checklist has been updates with the results. Large timing violations have been discussed mainly the  Macro2Flop violation that will vanish by implementing a Multicycle SRAM.  Multicycle SRAM is tested with Resiltech configuration.

Waiver page created and it is acceptable: Ncore 3.6.0 Timing Violations Waiver
