MemorySet: User can define maximum of 2 Memory Sets.

Each memory set contains groups of all the DMIs in the system.

For example. User defines 5 MEMORY sockets. 

This generates 5 DMIs. dmi0, dmi1, dmi2, dmi3, dmi4.

The user can define the 2 Memory Sets with the groups as follows.

MemorySet0 :  dmi0, dmi1, dmi2, dmi3 in dmg0

                       dmi4 in dmg1

MemorySet1:  dmi0, dmi1 in dmg2

                       dmi2, dmi3 in dmg3

                      dmi3 in dmg4

DynamicMemoryGroup: The groups under the MemorySet form the DynamicMemoryGroup.

Each DynamicMemoryGroup can have a size of 2, 4, 8 or 16 DMIs.

In the Above example, we have 5 DynamicMemoryGroups

Dmg0 has 4 DMIs    →  4 bit Interleaving

DMg1 has 1 DMI

Dmg2 has 2 DMIs    → 2 bit interleaving

Dmg3 has 2 DMIs    → 2 bit Interleaving

Dmg4 has 1 DMI

Taking into account all the groups, we need 4 bit DMI InterleavingFunction and 2 bit DMI InterleavingFunction. 


MemoryInterleavingFunction

If any group has 4 DMIs, we need to define 4 way MemoryInterleavingFunction.

We need 2 bits to determine the DMI.  User can specify maximum of 2 options for the bits. 

E.g. FourWayInterleavingFunction1 : "{7,8}"

      FourWayInterleavingFunction2 : "{13,14}"

If any group has 2 DMIs, we need to define 2 way MemoryInterleavingFunction.

we need 1 bit to specify the DMI.

E.g. TwoWayInterleavingFunction1: "{10}"

       TwoWayInterleavingFunction2: "{11}"
