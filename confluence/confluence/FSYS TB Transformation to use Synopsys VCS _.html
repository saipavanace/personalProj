HOW TO RUN SIMULATIONS (ncore3.4):

  1. New checkout: /engr/dev/tools/script/setup_concerto

  2. source SOURCEME ncore3.4

  3. To run with QUESTA simulator (runsim -sim QUESTA), add this step : export UVM_HOME=/engr/dev/tools/mentor/questa-2021.2_1/questasim/verilog_src/uvm-1.1d (in CSH: export UVM_HOME /engr/dev/tools/mentor/questa-2021.2_1/questasim/verilog_src/uvm-1.1d)

  4. Run command

    1. With questasim: runsim -e fsys -c <config_name> -t <test_name> -sim QUESTA

    2. With VCS : runsim -e fsys -c <config_name> -t <test_name>
