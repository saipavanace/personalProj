Maestro outputs DC synthesis scripts that are more suited to DCTopo/DCNXT. Designers who want to run zero wire load synthesis using these scripts will fail. Hence, I created a simple automation to build zero wire load synthesis for you.

Here are the steps you need to follow once you generate RTL from maestro.

  1. Copy zwld setup script to your synthesis scripts directory.

cp /scratch/smusini/zwld_synth_setup/7nm/buildRun_zwld_synth.csh   output/synthesis/DCNXT/scripts

  1. cd output/synthesis/DCNXT/scripts

  2. update your block name in the script or add all block names for conversion.

  3. source buildRun_zwld_synth.csh

This script builds zero wire load setup and also creates run_dc executable for you.

  1. Now, go into the block and source run_dc.

for example:

cd dmi_c

source run_dc

Example run: /scratch/smusini/devel/projects/zwld_synth/example2/output/synthesis/DCNXT/scripts/dmi_c
