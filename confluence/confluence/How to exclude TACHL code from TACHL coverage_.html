If you want to ignore some part of code simply write 

Ignore an else path

/* istanbul ignore else */
if (foo.hasOwnProperty('bar')) {
    // do something
}


Usually istanbul would complain about missing coverage for the else branch but it won't do so because of the comment

Ignore an if path

/* istanbul ignore if */
if (hardToReproduceError)) {
    return callback(hardToReproduceError);
}


If you want to exclude some part of the code for a single block [eg: exclude it just for Legato], you would use the following syntax:

/* istanbul ignore if env legato */

If you want to exclude some part of the code for multiple blocks [eg: exclude it for Legato, IOAIU 3.2 & CHIAIU 3.x], you would use the following syntax:

/* istanbul ignore if env legato, ioaiu_3p2, chiaiu_3px */
