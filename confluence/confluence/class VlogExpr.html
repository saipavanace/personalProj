Description

This is the topmost class in the class hierarchy of classes in the Verilog expression generator. This class is used to represent a Verilog expression.

Properties

An instance of the class will have the following data-properties, some of which are implemented as getters:

  1. width: Represents the width of the expression in bits. Read-only.

Interface (Methods)

The class provides the following interface methods: 

constructor ( width, expr, lastOp ) (not user-accessible)

Creates an instance of VlogExpr. width is the width of the expression and must be either zero or a positive ineger. expr is a string which has the verilog expression. lastOp is the last operation performed (also a string).

replicate ( n )

Returns a new object of type VlogExpr which is the original expression replicated by n times. n must be positive. When n = 1, the returned expression will be identical to original expression on which this method is called.

cat ( expr2, expr3, expr4, ... )

Returns a new object of type VlogExpr which is the original expression concatenated with expr2, expr3, expr4 and so on. expr2, expr3, expr4, ... are all expected to be of type VlogExpr. Zero-width expressions are removed from the concatenated result.

pad0 ( n )

Returns a new object of type VlogExpr, which is the original expression concatenated with n zeros on the MSB side. n is zero or a positive integer. If n === 0, the result is identical to the original expression.

pad1( n )

Same as pad0 except pads with ones instead of zeros.

pad0lower ( n )

Same as pad0 except pads zeros on the LSB side.

pad1lower ( n )

Same as pad1 except pads ones on the LSB side.

unaryOp ( op )

Returns a new object of type VlogExpr, which is the result of performing the specified unary operation - op. op is a single-character string and supports all unary Verilog operators.

binaryOp ( op, expr )

Returns a new object of type VlogExpr, which is the result of performing the specified binary operation, op on the input expression, expr. op is a single-character string and supports all binary Verilog operators.

ternaryOp ( condition, expr )

Return type: VlogExpr. Returns a conditional expression with the expression on which this is called being the default expression ( i.e. when condition is false ). This is equivalent to condition ? expr : this, where condition and expr are expected to be of type VlogExpr, and this is the expression on which the method is called (context).

assign ( expr, opts )

Return type: string. Returns a string containing a Verilog statement expression assigning the expression, expr to the expression on which this method is called. The parameter opts (options) is optional. It provides options for:

  1. WidthCorrection: Use the field 'widthCorrect' which is an object with further two fields one for RHS (rhs), and one for LHS (lhs) and can take values such as "pad0", "pad1", "trunc" (match the other expression with sufficient padding or truncation).
  2. Indentation: Use the field 'indent' Has fields such as tabs to separate the assignment operator with specified number of tabs.
  3. Assignment type: Field 'assign'. Has fields op and type. op is a string that can be either "=" (blocking) (default) or "<=" (non-blocking). type is a string that can be either "continuous" (default) or "procedural".
