Uncorrectable Error

When Uncorrectable Error Logging is Enabled: Check that the RTL CSR register has the uncorrectable error bit set to 1 and all other fields or xUESR, xUELR0 and xUELR1 when an uncorrectable error occurs.

#Check.DCE.Concerto.v3.7.UncorrectableError

2

When Uncorrectable Error Logging is Disabled: Check that the RTL CSR register has the uncorrectable error bit set to 0 when an uncorrectable error occurs.

3

When Error Interrupt Is Enabled: Check that the RTL CSR register has the error interrupt is asserted when an uncorrectable error occurs.

4

When Error Interrupt Is Disabled: Check that the RTL CSR register has the error interrupt is not asserted when an uncorrectable error occurs.

5

Mission Fault(Uncorrectable Error + resiliency)

#Check.DCE.Concerto.v3.7.MissionFault

7

8

When Error Interrupt Is Disabled: Check that the RTL CSR register has the error interrupt is not asserted when an uncorrectable error occurs.

9

Correctable Error

When Correctable Error Logging is Enabled: Check that the RTL CSR register has the uncorrectable error bit set to 1 and all other fields or xCESR, xCELR0 and xCELR1 when an correctable error occurs.

#Check.DCE.Concerto.v3.7.CorrectableError

10

When Correctable Error Logging is Disabled: Check that the RTL CSR register has the uncorrectable error bit set to 0 when an Correctable error occurs.

11

12

When Error Interrupt Is Disabled: Check that the RTL CSR register has the error interrupt is not asserted when an uncorrectable error occurs.

2. SRAM Input Flop For DM - useSramInputFlop

2.1 Feature

Feature as listed in the DCE micro-architecture document.

2.2 Stimulus

This feature does not require new stimulus generation and hence no additional sign-off criterion. However, it requires DCE TB updates and triage.

2.3 Functional Checks

This feature does not result in no new functional checks. However, requires that the DCE TB be updated

Update the DCE RTL internal probe monitors to accommodate P0+ pipeline stage.

Update the DCE scoreboard to accommodate P0+ pipeline stage.

2.4 Functional Coverage

This feature does not result in now new functional cover points.
