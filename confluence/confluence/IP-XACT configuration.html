When user Press "Generate RTL" button the IP-XACT file get generated.

The name of the file <project_name>.xml

The file comply with IEEE Std 1685-2014

The contain several sections with information about the IP block being emitted.

Bus Interfaces

The list of all Ncore sockets exposed outside Top level.

<ipxact:busInterfaces> [ <ipxact:busInterfaces> ]#

Depend on configuration this section can contain multiple AXI, ACE, ACE-Lite, OCP, APB protocol sockets.

Each socket <ipxact:busInterface> describes mapping of physical pins to the there logical roles.

note: Some ACE may contain AWUNIQUE signal mapping that is incompatible with old amba.com/AMBA4/ACE/r0p0_0 schema.

Memory Maps

<ipxact:memoryMaps> [ <ipxact:memoryMap> ]#

A memory map is defined for each CSR slave interface of a component.

The memoryMaps element contains a list of memoryMap elements. One per Ncore unit.

Model Ports

<ipxact:model><ipxact:ports> [ <ipxact:port> ]#

Contains the list of ports for Ncore top module.
