This is one of the JS functions returned by vlogGen() function, which can be found in the following path and file:
hw-lib/js/lib_utils.js

Base type

vlogExpr

Composition

None

Description

This is a factory function that returns an object that represents a Verilog constant along with functions to perform various operations on the constant. Every constant has a value and width associated with it. The width can be any number in the range 0 to 230. And value can be any value that can be represented with as many bits as they are stored as BigInts.

Usage

The factory function can be called using the following call signature:

vlogConst( width, value,  options )

where width is a positive integer that represents the width of the signal / bit-vector in bits, value is the value of the constant (type: number or BigInt), and options is an optional parameter with the following properties:

  1. radix: Specifies the radix or number base to represent the value in signal expressions. If not specified, it defaults to 10.

Interface

The object returned by the vlogConst( ) factory function provides the following properties:

Data

The returned object will have the following data-properties in additions to properties of the base type object (vlogExpr):

  1. value. type: BigInt. Represents the value of the constant.

Methods

The returned object will provide the following methods:

lShift ( n, truncWidth )

Return type: vlogExpr. Returns a vlogExpr object resulting from left-shifting (zeros padded from LSB) the constant. The first parameter, n, is the shift amount of type number or vlogSignal (variable shift). The second parameter, truncWidth, is optional and can be used to specify the width of the resulting expression. If not specified, the resulting width = width of the vlogSignal object that is shifted + n.

rShift ( n, truncWidth )

Return type: vlogExpr. Returns a vlogExpr object resulting from right-shifting (zeros padded from LSB) the constant. The first parameter, n, is the shift amount of type number or vlogSignal (variable shift). The second parameter, truncWidth, is optional and can be used to specify the width of the resulting expression. If not specified, the resulting width = width of the vlogSignal object that is shifted + n.

complement ( )

Return type: vlogConst. Returns a vlogConst object resulting from taking the one's complement of the constant on which this is called.
