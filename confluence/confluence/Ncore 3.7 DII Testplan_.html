1. Feature:

1.1 Skid Buffer Implementation.

The Skid Buffer inside the DII can be now implemented using SRAM. To enable the use of SRAM, the CMDOverflowBufInSRAM (should be turned on) when the overflow buffer is more than 256. (Overflow buffer size = n*SkidBufSize - n*SkidBufArb). When it is enabled, the memory protection cannot be NONE and should be either PARITY or SECDED.
