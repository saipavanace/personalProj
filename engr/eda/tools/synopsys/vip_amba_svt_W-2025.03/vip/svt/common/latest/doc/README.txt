SVT: Release Notes - V-2023.12-T-20240621
-------------------------------------------------------------------------

1. Introduction:

This version of SVT is basically the SVT V-2023.12 release combined with
a limited set of bug fixes. As such it is supplied with the existing SVT
V-2023.12 documentation. The Bug Fixes section provides brief summaries
of the bug fixes which are included in this release.

-------------------------------------------------------------------------

2. Bug Fix:

- Updated printing of BITVEC field data in IEEE UVM versions to resolve the truncation of data above 64 bits. 
*(STAR #P80002216-85856)*

-------------------------------------------------------------------------	

-------------------------------------------------------------------------

2. Enhancement:

- Added new enumerated values for 4.5GB memory depth & 2125 MHZ speed.
*(STAR P80002216-83721)*
